id (benzersiz kimlik) seçici

id özelliği html elemanına benzersiz bir kimlik tanımlaması yapar.id'lerde türkçe karakter kullanılmaz.id seçici kullanırken elemanın adını yazmak zorunda değilsiniz. # işaretinden sonra oluşturduğunuz id' nin ismini yazmanız yeterli. Aynı id ismini birden fazla kullanmak doğru değildir.

Örnek ;

#baslik {
color: red;
}
p#paragraf {
color: orange;
}
