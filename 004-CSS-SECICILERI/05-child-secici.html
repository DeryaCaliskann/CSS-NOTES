child (çocuk) seçici

Ana elemanın bir altındaki eleman çocuk elemandır. Ana tag'dan sonra büyüktür ">" işareti koyup tag ismi yazılarak seçilir.

Çocuk seçici id ve Class seçiciyle beraber de kullanılabilir. Tüm Seçiciciler iç içe kullanılabilir.

Örnek ;

div>p>span {
text-decoration: underline;
}
div>p#kirmiziyazi {
color: red;
}
