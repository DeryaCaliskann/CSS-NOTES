attribute (özellik) seçici

Belirtilecek olan özelliğe sahip html elemanını seçer.

Attribute (özellik) seçicileri için filtreler

  • Tam değer filtresi; [özellik adı="değer"] (Özellik değerinin tam olarak yazılması gerekiyor. i harfi büyük harf küçük harf duyarlılığına bakma demek.)
  • İçeren tam değer filtresi; [özellik adı~="değer"](Özellik değerinin yazılan kısmının başındaysa sonunda, sonundaysa başında, ortadaysa her iki tarafında boşluk olması gerekli.)
  • İçeren değer filtresi; [özellik adı*="değer"] (Kelimenin başında ve sonunda boşluk olması gerekmez.)
  • Başlayan değer filtresi; [özellik adı^="değer"] (Örneğin özellik değer ismi b harfiyle başlayanları seçer.)
  • Tireli ve tiresiz başlayan değer filtresi; [özellik adı|="değer"](Genelde lang özelliği için kullanılır Örneğin sadece en yada en- olanları seçer)
  • Biten değer filtresi; [özellik adı$="değer"](Belirtilecek olan özelliğe ve belirtilecek olan değere bitişine Html elementini seçer.

Örnek ;

ul li {
font-size: large;
}

li[name="tamdeger" i] {
color: red;
}

li[name~="iceren"] {
color: blue;
}

li[title*="iceren"] {
color: purple;
}

li[name^="b"] {
color: green;
}

li[lang|="en-us"] {
color: brown;
}

li[title$="ten"] {
color:yellow;
}
