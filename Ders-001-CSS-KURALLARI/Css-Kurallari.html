CSS'İN MUTLAK ŞART VE KURALLARI:

  • Css kodları sayfa içerisinde kullanılıyor ise mutlaka style tag'ı(etiketi) içerisinde yazılmalıdır.
  • Css kodları sayfa içerisinde Style tagı içerisinde kullanılıyor ise her seçici { } (süslü parantez) içerisinde yazılmalıdır.
  • Css kodları sayfa içerisinde satır içi kullanılıyor ise mutlaka style global özelliği içerisinde yazılmalıdır.
  • Css kodları sayfa içerisinde ve satır içi class(sınıf) şeklinde kullanılıyor ise mutlaka class global özelliği içerisine dahil edilmelidir.
  • Css'de özellik ve değer arasında mutlaka : (ikinokta) karakteri eklenmelidir.
  • Css'de özellik ve değer kullanıldıktan sonra mutlaka ; (noktalı virgül) ile sonlandırılmalıdır.
  • Css'de birden fazla tag'a erişilmek isteniyorsa mutlaka taglar arası , (virgül) eklenmelidir.
  • Sayfa harici bir Css dosyası dahil edilmek isteniyor ise link html tagı kullanılmalıdır.
  • Harici Css Dosyalarında karakter seti tanımı: @charset"utf-8" ; (Tanımlama en üstte olmalıdır. Birden fazla varsa en üstteki kabul edilir. Sadece harici Css dosyaları için gereklidir. Dahili Css kodları için gerekmez.)

Sayfa Düzeyinde Stil Tanımlaması;

Css kodlarının Html sayfası içerisinde, head tag'ları içerisindeki, style tag'ları arasında yazılmasıdır.

Element Düzeyinde Stil Tanımlaması;

Html sayfası içerinde, tag'ların içerisinde, style özelliği kullanılarak yazılan Css kodarıdır.(Yani satır içi Css. Çok sık kullanılmaz.)
