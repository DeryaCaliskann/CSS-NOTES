border-image-repeat

Html elementinin kenarlık resminin tekrarlanma özelliğini belirler.

değerler ;

  • repeat ; tekrarla
  • round ; yuvarla
  • stretch ; esnet
  • initial ; html elementi varsayılan değere döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

border-image-repeat:stretch;

Örnek-2 :

border-image-repeat:repeat;

Örnek-3 :

border-image-repeat:round;

Örnek-4 :

border-image-repeat:initial;

Örnek-5 :

border-image-repeat:round;

border-image-repeat:inherit;
