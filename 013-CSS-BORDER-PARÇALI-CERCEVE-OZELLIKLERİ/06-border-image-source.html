border-image-source

Html elementinin kenarlık resmini belirler. Prefix olmadan kullanabiliriz.

değerler ;

  • None ; değer yok
  • İmage ; resim değeri
  • initial ; varsayılan değere döner.
  • inherit ; üst elementinin değerini alır.

Örnek-1 :

border-image-source: url(../img/desen1.jpg);

Örnek-2 :

border-image-source:none;

Örnek-3 :

border-image-source: initial;

Örnek-4 :

border-image-source: url(../img/desen1.jpg);

border-image-source: inherit;
