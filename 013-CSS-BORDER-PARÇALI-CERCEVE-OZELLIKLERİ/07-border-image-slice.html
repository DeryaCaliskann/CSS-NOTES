border-image-slice

Html elementinin kenarlık resminin dilimleme özelliklerini belirler.

değerler ;

  • Length ; uzunluk
  • % ; yüzde
  • fiil ; doldurma değeri
  • initial ; html elementi varsayılan değere döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

border-image-slice: 30;

Örnek-2 :

border-image-slice:15%;

Örnek-3 :

border-image-slice:33%fiil ;

Örnek-4 :

border-image-slice:20% 25%fiil ;

Örnek-5 :

border-image-slice:initial ;

Örnek-6 :

border-image-slice: url(../img/desen1.jpg);

border-image-slice: inherit;
