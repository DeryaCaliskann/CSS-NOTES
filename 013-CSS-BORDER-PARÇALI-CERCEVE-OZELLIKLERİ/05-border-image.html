border-image

Html elementinin kenarlık resim özelliklerini belirler. Prefix'lerle beraber kullanılır.

parametreler ;

1- image (resim)

  • İmage;resim değeri
  • initial;varsayılan değere döner.
  • inherit;üst elementinin değerini alır.

2- slice(dilimleme)

  • length ; uzunluk değeri
  • % ; yüzde değeri

3- repeat (tekrarlama)

  • repeat ; tekrarla
  • round ; yuvarla
  • stretch ; uzat,gerdir,esnet

Örnek-1 :

border-image: url(../img/desen1.jpg);

Örnek-2 :

border-image: url(../img/desen1.jpg) 30 stretch;(esnet)
(Varsayılan değer stretch'dir.)

Örnek-3 :

border-image: url(../img/desen1.jpg) 33%;

Örnek-4 :

border-image: url(../img/desen1.jpg)25 30 10 20;
1.parametre;sağ üstten yükseklik
2. parametre;sağ üstten genişlik.
3. parametre;sağ alttan yükseklik.
4. parametre; sol üstten genişlik

Örnek-5 :

border-image: url(../img/desen1.jpg) 25 30;
1.parametre;sağ üstten yükseklik
2. parametre;sağ üstten genişlik.

Örnek-6 :

border-image: url(../img/desen1.jpg) 30 repeat;(tekrarla)

Örnek-7 :

border-image: url(../img/desen1.jpg) 30 round;(yuvarla)
