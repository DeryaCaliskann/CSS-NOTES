grid-template-rows özelliği

Izgara düzenini içerisindeki element veya elementlerin, oluşturulması gereken satır boyutu yüksekliğini belirler.

değerler ;

  • none ; değer yok (varsayılan)
  • auto ; otamatik değer alır
  • max-content ; maksimum içerik.
  • min-content ; minimum içerik.
  • length ; uzunluk
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
