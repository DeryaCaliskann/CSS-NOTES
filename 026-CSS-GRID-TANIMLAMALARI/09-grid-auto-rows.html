grid-auto-rows özelliği

Izgara düzenini içerisindeki element veya elementler için, oluşturulması gereken satırların varsayılan yüksekliğini belirler.

değerler ;

  • outo; otamatik değer alır(varsayılan)
  • max-content ; maksimum içerik.
  • min-content ; minimum içerik. En uzun kelimeye göre boyut alır. Satır boşluklardan kırılır.
  • length ; uzunluk değeri
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
