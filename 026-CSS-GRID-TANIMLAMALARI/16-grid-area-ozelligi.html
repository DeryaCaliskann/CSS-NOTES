grid-area özelliği

Izgara düzenini içerisindeki elementin tüm satır ve sütun işlemleri için alan yerleştirme özelliklerini belirler. Aynı zamanda ızgara düzenini içerisindeki elementin alan adı değerlerini belirler. (genellikle alan adı tanımlamak için kullanılır)

parametreler

  • grid-row-start ; satır başalangıcı
  • grid-column-start ;sütun başlangıcı
  • grid-row-end ; satır bitişi
  • grid-column-end ; sütun bitişi

    • değerler ;

    • areaname; alan adı
    • initial ; html elementi varsayılan biçimlendirme değerine döner.
