align-self özelliği

Izgara düzenini içerisinde bulunan ve seçilen element veya elementlerin dikey hizalama özelliklerini belirler.

değerler ;

  • auto; otamatik değer alır. (varsayılan)
  • stretch ; uzat / esne
  • center ; orta
  • start ; başlangıç
  • end ; bitiş
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
