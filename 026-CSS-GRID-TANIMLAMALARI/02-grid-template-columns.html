grid-template-columns özelliği

Izgara düzenini içerisindeki element veya elementler için, oluşturulması gereken sütun adedini ve sütun boyutu genişliğini belirler.

değerler ;

  • none ; değer yok (varsayılan)
  • auto ; otamatik değer alır
  • max-content ; maksimum içerik. İçerik kadar genişlik kaplar. Hiç bir şekilde içerik bölünmez. Alanın dışına taşabilir.
  • min-content ; minimum içerik. Minimum oluşturabileceği genişiliğe göre oluşturur. Boşluklardan aşağı kırar.
  • length ; uzunluk
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
