grid-auto-flow özelliği

değerler ;

  • row ; satır (varsayılan)
  • column ; sütun
  • row dense ; satır yoğunluğu
  • column dense ; sütun yoğunluğu
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
