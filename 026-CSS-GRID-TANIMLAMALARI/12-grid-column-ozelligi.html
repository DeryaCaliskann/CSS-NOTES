grid-column özelliği

Izgara düzenini içerisindeki elementin kaçıncı sütundan başlaması gerektiğini ve kaçıncı sütunda bitmesi gerektiğini belirler.

parametreler

  • grid-column-start ; başalandıç
  • grid-column-end ; bitiş

    • değerler ;

    • outo; otamatik değer alır(varsayılan)
    • column-line ; sütun çizgisi.
    • initial ; html elementi varsayılan biçimlendirme değerine döner.
