justify-content özelliği

Izgara düzenini içerisinde bulunan elementler tüm alanı kullamadığında elementlerin yatay hizalama özelliklerini belirler.

değerler ;

  • stretch ; uzat / esne (varsayılan)
  • center ; orta
  • start ; başlangıç
  • end ; bitiş
  • space-between ; boşluk arasında
  • space-around ; boşluk etrafında
  • space-evenly ; eşit boşluk
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
