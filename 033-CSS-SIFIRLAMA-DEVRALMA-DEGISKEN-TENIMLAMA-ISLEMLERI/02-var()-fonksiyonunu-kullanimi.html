var() fonksiyonunun kullanımı ;

Önceden tanımlanmış olan ve tüm css kodları içerisinde kullanılmak üzere belirli değerler içeren verilerin tutulduğu değişkenlerin çağırılma durumunu belirler.Bu bize sitelerimizi yaparken büyük kolaylık sağlar. Değişkenler ya belgenin kök dizinine tanımlanır (yani :root sahte sınıfına) ya da elementlerin içerisinde olduğu üst elemente tanımlanmalıdır. Aksi takdirde çalışmaz. Kök dizine tanımlamak daha mantıklıdır. Böyle bütün belge içerisinde kullanabiliriz. Değişken isimlerini anlamlı oluşturmak, kullanmak karışıklığa mahal vermemek açısından önemlidir.

örnek-1 ;

     :root {
        --kirmiziRenk:red;
        --maviBorder:2px solid blue;
        --kalinYazi:bold;
        --sariArkaplan:yellow;
        --genislik:600px;
        --yukseklik:100px;
        --yaziOrtalama:center;       
        }

     div {
        width: var(--genislik);
        height: var(--yukseklik);
        border: var(--maviBorder);
        color: var(--kirmiziRenk);
        text-align: var(--yaziOrtalama);
        font-weight: var(--kalinYazi);
        background-color: var(--sariArkaplan);
        }
