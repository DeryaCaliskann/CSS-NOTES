all özelliğinin kullanımı ;

direction ve unicode-bidi hariç diğer tüm css özelliklerini sıfırlanacağı veya devralınacağı durumunu belirler.

değerler ;

  • initial ;Belirtilecek olan html elementinin,tüm özellikleri varsayılan biçimlendirme değerine döner.
  • inherit ;Belirtilecek olan html elementinin, üst elementinden normalde devralınmayan tüm css özelliklerini devralır.
  • unset ; Belirtilecek olan html elementinin, üst elementinden normalde devralınmayan tüm css özelliklerini devralır. Fakat üst elementinde tanımlı herhangi bir özellik yoksa tüm css özellikleri varsayılan biçimlendirme değerine döner.

örnek-1 ;

Ortada ki elemente all:initial değeri verildiğinde direction ve unicode-bidi hariç bütün biçimlendirme değerleri varsayılan biçimlendirme değerine döner.
