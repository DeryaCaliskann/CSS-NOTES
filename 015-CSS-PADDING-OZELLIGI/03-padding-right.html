padding-right özelliğinin kullanımı

Html elementinin sağ dolgu (iç boşluk) özelliklerini belirler.

değerler ;

  • length ; uzunluk değeri ;
  • % ;yüzde değeri ;
  • initial; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ;html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

padding-right: 25px;

Örnek-2:

padding-right:5%;

Örnek-3:

padding-right:initial;

Örnek-4:

padding-right:inherit
