padding-left özelliğinin kullanımı

Html elementinin sol dolgu (iç boşluk) özelliklerini belirler.

değerler ;

  • length ; uzunluk değeri ;
  • % ;yüzde değeri ;
  • initial; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ;html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

padding-left: 25px;

Örnek-2:

padding-left:5%;

Örnek-3:

padding-left:initial;

Örnek-4:

padding-left:inherit
