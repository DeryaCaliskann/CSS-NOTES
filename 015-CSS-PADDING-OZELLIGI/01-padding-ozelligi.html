padding özelliğinin kullanımı

Html elementinin dolgu (iç boşluk) özelliklerini belirler. Yani içerikle border arasında kalan kısımdır. Bazı elementlerin varsayılan padding değerleri vardır. Bu yüzden önce (padding:0px) sıfırlanır sonra yeni değer verilir. Böyle yaparsak herhangi bir dolgu bozukluğuna maruz kalmayız.

değerler ;

  • length ; Uzunluk değeri ;
    • 4 parametreli: 1. üst 2. sağ 3. alt 4. sol
    3 parametreli:1. üst 2. sağ-sol 3.alt
    2 parametreli: 1.üst-alt 2. sağ-sol
  • % ;yüzde değeri ; 4,3,2 parametrelide kullanılabilir.
  • initial; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ;html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

padding: 25px;

Örnek-2 :

padding: 50px 30px 25px;

Örnek-3 :

padding: 50px 30px 25px;

Örnek-4:

padding: 50px 30px;

Örnek-5:

padding:5%;

Örnek-6:

padding:initial;

Örnek-7:

padding:inherit
