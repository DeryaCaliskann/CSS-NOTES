BORDER-COLOR ÖZELLİĞİ

Html elementinin kenarlık, çerçeve rengini belirler.

DEĞERLER

  • color; renk
  • transparent; transparan
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

ÖRNEKLER

Örnek-1

border-color:red

Örnek-2

border-color:transparent

Örnek-3

border-color:initial (varsayılan)

Örnek-4

p: border-color:blue > span: border-color;inherit (miras)

Örnek-5

border-color: blue yellow green red (1.parametre;üst 2.parametre;sağ 3.parametre;alt 4.parametre;sol saat yönünde gider)

Örnek-6

border-color: blue yellow green (1.parametre;üst 2.parametre;sağ-sol 3.parametre;alt)

Örnek-7

border-color: blue yellow (1.parametre;üst-alt 2.parametre;sağ-sol)
