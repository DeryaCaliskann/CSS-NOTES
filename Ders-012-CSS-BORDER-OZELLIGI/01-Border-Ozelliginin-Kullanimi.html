CSS'DE BORDER ÖZELLİĞİ

Html elementinin kenarlık, çerçeve özelliliklerini belirler.

PARAMETRELER ;

Length, Style , Color parametrelerini alır.

1 - LENGTH

Değerleri :

  • length ; uzunluk
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.
  • thin ; ince
  • medium ; orta
  • thick ; kalın

2 - STYLE

Değerleri :

  • none ; değer yok
  • hidden ; gizli
  • solid ; düz çizgi
  • double ; çift çizgi
  • dashed ; keşikli çizgi
  • dotted ;noktalı çizgi
  • groove ; içe dönük gölgeli çizgi
  • ridge ; dışa dönük gölgeli çizgi
  • inset ; içe dönük derin çizgi
  • outset; dışa dönük derin çizgi

3 - COLOR

Kenarlıkları renklerini belirler.

ÖRNEKLER

Örnek-1

border: 1px solid blue

Örnek-2

border: thin solid blue

Örnek-3

border: medium solid blue

Örnek-4

border: thick solid blue

Örnek-5

border: initial (varsayılan)

Örnek-6

span;inherit
span;inherit
span;inherit

Örnek-12

Örnek-13

Örnek-14
