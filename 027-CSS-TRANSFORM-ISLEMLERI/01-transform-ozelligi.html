tansform (dönüşüm) özelliği

Html elementinin 2D veya 3D dönüştürme özelliklerini belirler. Yapılan işlemler boyutlandırma için değil animasyonlar içindir. Özellikler eksi değerde alabilir.

değerler ;

  • none ; değer yok (varsayılan)
  • translate () ; Html elementinin x ve y ekseni doğrultusunda konumunu değiştirir.
  • translate3d () ; Html elementinin x , y ve z ekseni doğrultusunda konumunu değiştirir.
  • translateX () ; Html elementinin x ekseni doğrultusunda konumunu değiştirir.
  • translateY () ; Html elementinin y ekseni doğrultusunda konumunu değiştirir.
  • translateZ () ; Html elementinin z ekseni doğrultusunda konumunu değiştirir.
  • skew () ; Html elementini x ve y ekseni doğtultusunda eğer. ;
  • skewX () ; Html elementini x ekseni doğtultusunda eğer.
  • skewY () ; Html elementini y ekseni doğtultusunda eğer.
  • scale () ; Html elementini x ve y ekseni doğtultusunda boyutunu değiştirir.
  • scale3d() ; Html elementini x , y ve z ekseni doğtultusunda boyutunu değiştirir.
  • scaleX () ; Html elementini x ekseni doğtultusunda boyutunu değiştirir.
  • scaleY () ; Html elementini y ekseni doğtultusunda boyutunu değiştirir.
  • scaleZ () ; Html elementini z ekseni doğtultusunda boyutunu değiştirir.
  • rotate () ; Html elementini saat yönü doğrultusunda döndürür.
  • rotate3d () ; Html elementini x , y ve z ekseni doğrultusunda döndürür.
  • rotateX () ; Html elementini x ekseni doğrultusunda döndürür.
  • rotateY () ; Html elementini y ekseni doğrultusunda döndürür.
  • rotateZ () ; Html elementini z ekseni doğrultusunda döndürür.
  • perspective () ; Html elementinin perspektif görünüşünü belirler.
  • matrix () ; Html elementine altı farklı dönüşüm özelliği belirler. (scaleX(), skewY(), skewX(), scaleY(), translateX(),translateY())
  • matrix3d () ; Html elementine 4*4 dönüşüm özelliği belirler.( https://codepen.io/fta/pen/ifnqH )
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
