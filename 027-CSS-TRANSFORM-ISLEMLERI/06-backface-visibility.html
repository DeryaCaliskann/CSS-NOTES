backface-visility özelliği

Belirtilen Html elementinin dönüştürme geçiş veya animasyon işlemlerinde arka yüzeyinin görüntülenme veya gizlenme özelliklerini belirler.

parametreler ;

  • x ; varsayılan 50%
  • y ; varsayılan 50%

değerler ;

  • visible; göster
  • hidden; gizle
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

transform: rotateY(180deg);
backface-visibility: visible;

transform: rotateY(180deg);
backface-visibility: hidded;
