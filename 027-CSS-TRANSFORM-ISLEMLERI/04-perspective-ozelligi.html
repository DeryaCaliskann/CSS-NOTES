perspective özelliği

Html elementinin 3d dönüştürme sonrasındaki perspektif derinliği özelliklerini belirler.

değerler ;

  • none ; değer yok(varsayılan)
  • length ; uzunluk
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

transform: rotateX(45deg);
perspective: none;
Merhaba Dünya

örnek-2 ;

transform: rotateX(45deg);
perspective: 150px;
Merhaba Dünya

örnek-3;

perspective: 100px;
Merhaba Dünya

örnek-4;

transform: rotateY(45deg);
perspective: 150px;
Merhaba Dünya

örnek-5;

transform: rotateY(-45deg);
perspective: 150px;
Merhaba Dünya
