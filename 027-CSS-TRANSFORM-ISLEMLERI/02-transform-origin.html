tansform-origin özelliği

Html elementinin dönüştürme sonrasındaki konum özelliklerini belirler.

parametreler ;

  • x ; 50% varsayılan değer
  • y ; 50% varsayılan değer
  • z ; 0% varsayılan değer

değerler ;

  • length ; uzunluk değeri
  • % ; yüzde değeri
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

transform: rotate(40deg);
transform-origin: 50% 50% 0;
