perspective-origin özelliği

Html elementinin 3d dönüştürme sonrasındaki perspektif derinliği konum özelliklerini belirler.

parametreler ;

  • x ; varsayılan 50%
  • y ; varsayılan 50%

değerler ;

  • length ; uzunluk
  • % ; yüzde
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

transform: rotateX(45deg);
perspective: 150px;
perspective-origin: 50% 50%;
