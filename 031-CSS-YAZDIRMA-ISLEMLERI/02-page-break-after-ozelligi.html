page-break-after özelliğini kullanımı;

Sayfa yazdırma işlemlerinde belirtilecek olan Html elementinden sonra sayfa sonlandırma durumunu belirler.

değerler ;

  • auto ; otomatik değer alır. Yani sonlandırma yapmadan devam eder. (varsayılan)
  • always ; daima demek. Belirtilen elementten sonra sayfayı sonlandırır.
  • initial ; html elementi varsayılan biçimlendirme değerine döner.

Örnek :

 .sonlandir {
            page-break-after: always;
            }
