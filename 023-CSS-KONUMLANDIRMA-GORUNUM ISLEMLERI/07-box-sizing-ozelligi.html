box-sizing özelliği

Html elementinin genişlik ve yükseklik değerlerinin ne şekilde hesaplanması gerektiğini belirler. prefixlerle beraber kullanılır.

değerler ;

  • content-box ; İçerik kutusu (varsayılan)
    • sol kenarlık genişliği + sol dolgu genişliği + içerik genişliği +sağ dolgu genişliği+sağ kenarlık genişliği = genişlik
    • üst kenarlık genişliği + üst dolgu genişliği + içerik yüksekliği + alt dolgu yüksekiği+ alt kenarlık yüksekliği =yükseklik
  • border-box ; Kenarlık kutusu
    • içerik genişliği = genişlik
    • içerik yüksekliği = yükseklik
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır
sol kenarlık genişliği + sol dolgu genişliği + içerik genişliği +sağ dolgu genişliği+sağ kenarlık genişliği = genişlik
üst kenarlık genişliği + üst dolgu genişliği + içerik yüksekliği + alt dolgu yüksekiği+ alt kenarlık yüksekliği = yükseklik

Örnek-1 ;

10px + 50px + 500px + 50px+ 10px = 620px genişlik
10px + 50px + 300px + 50px + 10px = 420px yükseklik
content-box

Örnek-2 ;

500px = genişlik
300px = yükseklik
border-box
