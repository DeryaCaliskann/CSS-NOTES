box-sizing özelliği

Html elementinin genişlik ve yükseklik değerlerinin ne şekilde hesaplanması gerektiğini belirler. prefixlerle beraber kullanılır.

değerler ;

  • content-box ; İçerik kutusu (varsayılan)
  • border-bax ; Kenarlık kutusu
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

content-box

sol kenarlık genişliği + sol dolgu genişliği + içerik genişliği +sağ dolgu genişliği+sağ kenarlık genişliği = 620px

üst kenarlık genişliği + üst dolgu genişliği + içerik yüksekliği + alt dolgu yüksekiği+ alt kenarlık yüksekliği = 420px

Örnek-2 ;

border-box

içerik genişliği = 500px

içerik yüksekliği = 300px
