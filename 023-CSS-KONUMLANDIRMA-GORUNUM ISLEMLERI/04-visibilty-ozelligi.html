visibility özelliğinin kullanımı

Html elementinin görüntülenme ve gizlenme özelliklerini belirler.

değerler ;

  • visible ; görünür. Element gösterilir (varsayılan değer)
  • hidden ; gizli. Element gösterilmez ve elementin kapladığı alan değerleri korunarak boş bir alan oluşur.
  • collapse ; bitişik. Element gösterilmez ve elementin kapladığı alan değerleri korunarak boş bir alan oluşur fakat element bit tablo elementi ise kapladığı alan değerleri korunmaz.
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; Html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

visibility:visible; -1
visibility:visible; -2
visibility:visible; -3

Örnek-2 ;

visibility:hidden; -1
visibility:hidden; -2
visibility:hidden; -3

Örnek-3 ;

visibility:collapse; -1
visibility:collapse; -2
visibility:collapse; -3

Örnek-4 ;

visibility:collapse; -1
