z-index özelliği

Konumlandırılmış bir Html elementinin katman /yığın sırasını belirler.

değerler ;

  • auto; otomatik değer alır. Herhangi bir farklılık olmaz. Elementlerin dizilim sırasına göre yer alır.
  • number; rakam değeridir. En büyük rakam değeri daima en üstte olacaktır.
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

background-color: red; / z-index:1;
background-color: gray; / z-index: 2;
background-color: yellow; / z-index:3;
