display özelliğinin kullanımı

Html elementinin görüntülenme özelliklerini belirler.

değerler ;

  • inline ; varsayılan değerdir. Html elementi satır düzeyli element olarak kabul edilir ve görüntülenir. Elemente içerik alanından fazla genişlik ve yükseklik atanmasına izin verilmez.
  • block ; Html elementi blok düzeyli element olarak kabul edilir ve görüntülenir.
  • inline-block ; Html elementi hem satır düzeyli hem blok düzeyli element olarak kabul edilir ve görüntülenir.Elemente içerik alanından fazla genişlik ve yükseklik atanmasına izin verir.
  • none ; değer yok. Html elementini görünmez yap demek.
  • run-in ; Html elementi ya satır düzeyli yada blok düzeyli element olarak kabul edilir ve görüntülenir. Elementin kendisi satır düzeyliyse satır, blok düzeyliyse blok düzeyli element olarak kabul edilir.
  • list-item ; Html elementi liste elementi olarak kabul edilir ve görüntülenir.
  • table ; Html elementi tablo elementi (table) olarak kabul edilir ve görüntülenir.
  • table-row ; Html elementi tablo satır (tr) elementi olarak kabul edilir ve görüntülenir.
  • table-cell ; Html elementi tablo sütün (td) elementi olarak kabul edilir ve görüntülenir.
  • table-caption ; Html elementi tablo kısa başlık (caption) olarak kabul edilir ve görüntülenir.
  • table-row-group ; Html elementi tablo gövde elementi (tbody) olarak kabul edilir ve görüntülenir.
  • table-header-group ; Html elementi tablo içi başlık elementi (thead) olarak kabul edilir ve görüntülenir.
  • table-footer-group ; Html elementi tablo içi altlık (tfoot) olarak kabul edilir ve görüntülenir.
  • table-column ; Html elementi tablo içi sütun ortak özellikleri belirleme elementi (col) olarak kabul edilir.
  • table-column-group ; Html elementi tablo içi sütun ortak özellikleri belirleme grubu elementi (colgroup) olarak kabul edilirve görüntülenir.
  • inline-table ; Html elementi satır düzeyli tablo olarak kabul edilir ve görüntülenir.
  • flex ; Html elementi blok düzeyli esnek element olarak kabul edilir ve görüntülenir.
  • inlane-flex ; Html elementi satır düzeyli esnek element olarak kabul edilir ve görüntülenir.
  • grid ; Html elementi blok düzeyli ızgara element olarak kabul edilir ve görüntülenir.
  • inline-grid ; Html elementi satır düzeyli ızgara element olarak kabul edilir ve görüntülenir.
  • initial ; Html elementi varsayılan biçimlendirme değerine döner. (her özellikte çalışmaz)
Not: Satır kapsamlı elementler sadece içerikleri kadar genişlik ve yükseklik kaplayabilirler.

Örnek-1 ;

1-display:inline;
2-display:inline;
3-display:inline;


Not:div blok düzeyli bir elementken display:inline özelliği ile satır düzeyli bir element gibi görüntülenir.Genişlik ve yüksekli değeri alamaz.

Örnek-2 ;

1-display:block; 2-display:block; 3-display:block;
Not:span satır düzeyli bir elementken display:block özelliği ile blok düzeyli bir element gibi görüntülenir. Genişlik ve yüksekli değeri alabilir.

Örnek-3 ;

1-display:inline-block; 2-display:inline-block; 3-display:inline-block;

Not:inline-block özelliği ile satır içi bir elemente genişlik ve yükseklik verebliriz. Hem satır düzeyinin hem blok düzeyinin özelliklerinden faydalanıyor.

Örnek-4 ;

1-display:none; 2-display:none; 3-display:none;

display:none özelliği ile elemente görünme diyoruz.

Örnek-5 ;

1-display:run-in; 2-display:run-in; 3-display:run-in;
Not: Elementin kendisi satır düzeyliyse satır, blok düzeyliyse blok düzeyli element olarak kabul edilir

Örnek-6 ;

    1-display:list-item; 2-display:list-item; 3-display:list-item;
Not: span yada herhangi bir elemeti liste itemi (li) gibi gösterir.
