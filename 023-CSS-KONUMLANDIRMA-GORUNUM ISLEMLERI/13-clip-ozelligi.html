clip özelliği

absolute (mutlak/kesin) değeri ile konumlandırılmış bir Html elementinin kırpma/maskeleme özelliklerini belirler.

değerler ;

  • auto ; otamatik değer alır(varsayılan). Herhangi bir değişiklik olmaz.
  • shape ; şekil rect( ) fonksiyonu kullanır. (rectangle : dikdörtgen)
  • initial; html elementi varsayılan biçimlendirme değerine döner

Örnek-1 ;

clip:auto;
