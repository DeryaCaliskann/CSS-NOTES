object-position özelliği

Html elemenin boyutlandırılmış alanına sığmayacak kadar büyük resim veya video içeriklerinin opject-fit özelliği kullanarak yeniden boyutlandırma sorasındaki resmin veya videonun konum özelliklerini belirler.

parametreler ;

  • horizontal value ; yatay değer
  • vertical value ;dikey değer

değerler ;

  • length ; uzunluk
  • % ; yüzde
  • center ; orta
  • top ; üst
  • right ; sağ
  • bottom ; alt
  • left ; sol
  • initial; html elementi varsayılan biçimlendirme değerine döner

Örnek-1

object-fit: cover;
object-position: 50px 100px; /*1-yatay 2-dikey*/

Örnek-2

object-fit: cover;
object-position: 50% 50%; /*1-yatay 2-dikey*/

Örnek-3

object-fit: cover;
object-position:center;

Örnek-4

object-fit: cover;
object-position: left;

Örnek-5

object-fit: cover;
object-position: right

Örnek-6

object-fit: cover;
object-position: left top;

Örnek-7

object-fit: cover;
object-position: right top;

Örnek-8

object-fit: cover;
object-position: center center;

Örnek-9

object-fit: none;
object-position: right bottom;
