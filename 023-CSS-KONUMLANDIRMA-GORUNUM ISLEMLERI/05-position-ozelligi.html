position özelliğinin kullanımı

Html elementinin konumlandırma özelliklerini belirler. Html elementinin blok yada satır düzeyli olmasının sayfa akışına etkileri vardır.

değerler ;

  • statik ; durağan demek.(varsayılan) Web sayfasını elementlerinin normal halidir.
  • relative ; görecli.
  • absolute ; mutlak / kesin.
  • fixed ; sabit.
  • sticky ; yapışkan
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; Html elementi üst elementinin biçimlendirme değerini alır.

diğer parametreler ;

  • top ; konumlandırılmış bir Html elementinin üstten hizalama konumunu belirler
  • right ; konumlandırılmış bir Html elementinin sağdan hizalama konumunu belirler
  • bottom; konumlandırılmış bir Html elementinin alttan hizalama konumunu belirler
  • left; konumlandırılmış bir Html elementinin soldan hizalama konumunu belirler
    • Değerleri;
    • auto ; otamatik değer alır
    • length ; uzunluk
    • initial ; varsayılan değere döner.

Örnek-1 ;

position: static;

Not:Elementin pozisyonu statik iken left right vs. komutlarları birşey değiştirmez. Konumlandırma yapamayız.

Örnek-2 ;

float: right;
position: relative;
left: 300px;

Not:Elementin pozisyonu relative iken left, right, top, bottom komutlarıyla hizalayabiliriz. eksi değerde alabilir.

Örnek-3 ;

float: right;
position: relative;
right: 300px;





Örnek-4 ;

float: right;
position: relative;
left: -300px;







Not: Eksi değer verebiliriz.

Örnek-5;

position: absolute;
right:200px;





Not:Hiç bir şeye bakılmaksızın verilen konuma konumlandırılır.

Örnek-6;

position: relative;




Örnek-7;

position:absolute;
position:absolute;






Not: absolite'de ne konum verildiyse onu çalıştırır. Üstünde element varmı bakmaz verilen konuma yerleşir.

Örnek-8;

2-position:absolute;
Not: Dıştaki elemente relative değeri verilirse, içerdeki kutu absolute olduğunda dış elemente göre konumlanır.






Örnek-9;

2-position:absolute;
Not: Dıştaki elemente relative değeri verilirse, içerdeki kutu absolute olduğunda dış elemente göre konumlanır.
