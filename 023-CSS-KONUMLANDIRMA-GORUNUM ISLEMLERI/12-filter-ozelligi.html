filter özelliği

Html elementine uygulanacak olan efekt özelliklerini belirler. prefix tanımlamak gerekir.

değerler ;

  • none ; değer yok
  • blur( ) ; bulanıklık (pixel değer alır)
  • brightness( ) ;parlaklık (0% ile %100 arası değer alır)
  • contrast( ) ; kontrast (0% ile %100 arası değer alır)
  • drop-shadow( ); gölge düşürme. Eksi değer dealabilir
    • horizontal shadow ; yatay gölge
    • vertical shadow ; dikey gölge
    • blur spread ; bulanıklık yayma
    • color ; renk
  • grayscale( ) ; gri tonlama (%0 ile %100 arası değer alır)
  • hue-rotate( ) ; renk döndürme (0 ile 360 derece arası değer alır)
  • invert( ) ; ters çevirme (%0 ile %100 arası değer alır)
  • opacity( ) ; opaklık / şeffaflık (%0 ile %100 arası değer alır)
  • saturate( ) ; doyma (%0 ile %100 arası değer alır)
  • sepia( ); kahverengi ağırlıklı antik tonlama (%0 ile %100 arası değer alır)

Örnek-1 ;

filter: none;

Örnek-2 ;
