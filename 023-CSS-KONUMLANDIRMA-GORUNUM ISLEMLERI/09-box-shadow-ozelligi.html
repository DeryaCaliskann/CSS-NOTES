box-shadow özelliği

Html elementinin gölgelendirme özelliklerini belirler.Prefix'lerle beraber kullanılır.

parametreler ;

  • horizontal shadow ; yatay gölge
  • vertical shadow ; dikey gölge
  • blur spread ; bulanıklık yayma
  • spread radius ; yayma yarıçapı
  • color: renk

değerler;

  • none: değer yok
  • length; uzunlukdeğeri
  • inset ; ilave (içerisinde)
  • initial; html elementi varsayılan biçimlendirme değerine döner

Örnek-1 ;

box-shadow: none;

Örnek-2 ;

box-shadow: 5px 5x;

Örnek-3 ;

box-shadow: -5px -5x;

Örnek-4;

box-shadow: 5px 5x grey;

Örnek-5;

box-shadow: 5px 5x 5x grey;

Örnek-6;

box-shadow: 5px 5x 5x 5px grey;

Örnek-7;

box-shadow: 5px 5x 5x 1px grey inset;

Örnek-8;

5px 5px red,5px 5px blue,5px 5px yellow;

Örnek-9;

5px 5px red,5px 5px blue,5px 5px yellow;
