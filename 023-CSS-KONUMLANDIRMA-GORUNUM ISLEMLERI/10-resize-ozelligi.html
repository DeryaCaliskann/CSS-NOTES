resize özelliği

Html elementinin kullanıcı tarafından yeniden boyutlandırılabilir olup olmadığı durumunu belirler. divlerde vs. kullanırken overflow değeri vermemiz gerekiyor. textarea'a vermemize gerek yok.

değerler ;

  • none ; değer yok
  • both ; her ikiside
  • horizontal ; yatay
  • vertical; dikey
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

resize:none;

Örnek-3;

resize:both;

Örnek-4;

resize:horizontal;
