opacity özelliği

Html elementinin opaklık / şeffaflık özelliklerini belirler (0.0 en şeffaf değer / 1.0 şeffaflık yok.). Her tarayıcıda sorunsuz çalışması için bir takım prefix'ler tanımlamak gerekir. Opacity özelliğinde element ve içindeki tüm değerler de şefffalaşır. Sadece arka plana şeffalık vermek istiyorsak rgba renk uzayını kullanabiliriz.

değerler ;

  • number ; rakam değeri
  • initial; html elementi varsayılan biçimlendirme değerine döner

Örnek ;

opacity: 1.0;
opacity: 0.8;
opacity: 0.6;
opacity: 0.4;
opacity: 0.2;
opacity: 0.0;
