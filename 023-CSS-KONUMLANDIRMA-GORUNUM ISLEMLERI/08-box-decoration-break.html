box-decoration-break özelliği

Html elementinin kenarlık / çerçeve özelliklerinin kutu parçalandığındaki satır veya sayfa sonları için davranış özelliklerini belirler. Satır düzeyli elementler için kullanılır. prefixlerle beraber kullanılır. prefix tanımlamadan çalışmıyor.

değerler ;

  • slice ; dilimle
  • clone ; copyala
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

box-decoration-break: slice;
