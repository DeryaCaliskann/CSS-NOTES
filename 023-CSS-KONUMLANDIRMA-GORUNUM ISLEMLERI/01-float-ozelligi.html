float özelliğinin kullanımı

Html elementinin bulunduğu konumdan kayma özelliklerini belirler.

değerler ;

- none; değer yok (varsayılandır. kayma değeri yok)

- left ; sola kay

- right ; sağa kay

- initial; html elementi varsayılan biçimlendirme değerine döner

- inherit; html elementi üst elementini biçimlendirme değerini alır

float: none;
