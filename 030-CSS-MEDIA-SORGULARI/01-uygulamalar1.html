@media sorguları kullanım örnekleri ;

örnek-1 ;

Not :Sadece ekran cihazlarında metin rengi kırmızı olur

@media only screen {
.ornek1 {
color:red;
}
}

örnek-2 ;

Not :Sadece yazdırıken (baskı makinelerinde) metin rengi kırmızı olur

@media only print {
ornek2 {
color:red;
}
}

not; ekran çıktısını görmek için ctrl+p

örnek-3 ;

Not :tüm chazlar için metnin stili italik olur.

@media only all {
.ornek3 {
font-style:italic;
}
}

örnek-4 ;

Not : ekranda ve(,) yazdırırken font-size' 15px olur.

@media only screen, print {
ornek4 {
font-size:15px;
}
}

not; ekran çıktısını görmek için ctrl+p

örnek-5;

Not :not (değil)operatörü ile kullanım.(ekran hariç tüm cihazlarda arka plan rengi sarı ol demek.)

@media not screen {
ornek5 {
backgroun-color:yellow;
}
}

not; ekran çıktısını görmek için ctrl+p

örnek-6;

Not :ekran genişliği en az 1200 px ve üzerindeyse mavi ol demek. 1200px altına indiğinde renk siyah olur.

@media only screen and (min-width:1200px) {
.ornek6 {
color:blue;
}
}

örnek-7;

Not :ekran genişliği en az 400 px ve en fazla 800px genişliğindeyse yeşil ol demek. Yani 400px ve 800px arası metin rengi yeşil olur, bunun dışındaki genişliklerde siyaha döner.

@media only screen and (min-width:400px) and (max-width:800px) {
.ornek7 {
color:green;
}
}

Not :ekran yüksekliği 200px ve 400px arasındaysa mor ol demek.

.ornek8{
color:purple
}
}
