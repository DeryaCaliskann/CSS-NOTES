text-oriantation

Metnin writing-mode özelliği ile belirtilmiş hizalama yönü özelliğinin döndürme durumunu belirler. prefixlerle kullanılır.

değerler ;

  • Mixed; Karışık (varsayılan değerdir)
  • Upright; Dik
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

text-orientation: mixed;

Örnek-2 ;

text-orientation: upright;
