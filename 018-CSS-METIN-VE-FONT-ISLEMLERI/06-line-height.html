line-height

Metin bloğunda bulunan satırların yüksekliğini belirler.

değerler ;

  • Normal; normal değer.
  • Number; rakam değeri.
  • Length; Uzunluk değeri.
  • %; yüzde değeri.
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.
