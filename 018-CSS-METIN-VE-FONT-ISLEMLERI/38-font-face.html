@font-face

Metinler için sistemde yüklü olmayan fontların kullanılabilme durumunu belirler.

değerler ;

  • font-family; metnin yazı tipi ailesi.
  • local; yerel
  • src; dosya adresi
  • font-style; metnin yazı tipi sitili
  • font-weight; metnin yazı tipi kalınlığı.

font dosya türleri

  • eot; internet explorer - microsof edge
  • woff2; chrome - firefox - opera
  • woff; tüm tarayıcılar
  • ttf & otf; tüm tarayıcılar
  • svg; chrome - safari - opera
