word-wrap

Html elementinin boyutlandırılmış alanına sığmayacak kadar büyük kelimenin satır kırılma özelliklerini belirler.

değerler ;

  • normal; normal (varsayılan değer)
  • break-word; kelimeyi kırar
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

word-wrap: normal;
