font-style

Metnin yazı tipi sitilini belirler

değerler ;

  • normal; normal
  • italic; italik
  • oblique; eğik (italikle aynı görünür. bazı fontların italik desteği olmadığında kullanılır)
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

font-style: normal;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci doloremque laborum voluptatibus reprehenderit quibusdam similique quae magni necessitatibus ipsam, qui sapiente
