writing-mode

Metnin veya içeriğin hizalama yönü özelliklerini belirler. prefixlerle kullanılır.

değerler ;

  • Horizontal-tb; yatay sekme. (varsayılan değer)
  • Vertical-lr; Dikey (sol).
  • Vertical-rl; Dikey (sağ).
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

writing-mode: horizontal-tb;

Örnek-2 ;

writing-mode: vertical-lr;

Örnek-3 ;

writing-mode: vertical-rl;
