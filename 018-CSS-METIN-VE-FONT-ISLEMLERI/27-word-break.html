word-break

Html elementinin boyutlandırılmış alanının sonuna sığmayacak kadar büyük kelimenin satır kırılma özelliklerini belirler.

değerler ;

  • normal; normal (varsayılan değer)
  • break-all; kelimeyi harflerden kırar
  • keep-all; kelimeyi boşluklardan kırar
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

word-break: normal;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci doloremque laborum voluptatibus reprehenderit quibusdam similique quae magni necessitatibus ipsam, qui sapiente mollitia, quis sit corporis molestias id tenetur, voluptate aliquam omnis saepe autem odit cum!

Örnek-2 ;

word-break: break-all;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci doloremque laborum voluptatibus reprehenderit quibusdam similique quae magni necessitatibus ipsam, qui sapiente mollitia, quis sit corporis molestias id tenetur, voluptate aliquam omnis saepe autem odit cum!

Örnek-3 ;

word-break: keep-all;
