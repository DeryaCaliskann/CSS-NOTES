text-overflow

Html elementinin boyutlandırılmış alanına sığmayacak kadar büyük metnin taşma özelliklerini belirler.

değerler ;

  • clip; metni kırpar
  • ellipsis; metni kırpar ve üç nokta (...) ekler
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

text-overflow: clip;
