counter özellikleri ile birlikte counters() fonksiyonunun kullanımı

counter-reset;

Bir yada fazla sayaç oluşturmak veya sıfırlamak için kullanılır.

counter-increment;

Bir yada fazla sayacın değerini arttırmak için kullanılır.

counters () ;

Oluşturulmuş olan ve iç içe geçmiş sayacın değerini Html elementine eklemek için kullanılır.

counters () Parametreleri ;

  • conuter name; sayaç adı
  • style-type değerleri;
    • decimal; ondalık sayı
    • decimal-leading-zero; ondalık sayı. Tek haneli sayılarda 0 takısı eklenir
    • lower-alpha; küçük harf
    • upper-alpha; büyük harf
    • lower-roman; küçük harfli roma rakamı.
    • upper-roman; büyük harfli roma rakamı.

Örnek:

ul {
list-style-type: none;
counter-reset: sayac;
}
li::before {
counter-increment: sayac;
content: counters(sayac," . ",lower-alpha) " - ";
}
