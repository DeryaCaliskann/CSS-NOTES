overflow-wrap özelliği

Html elementinin boyutlandırılmış alanına sığmayacak kadar büyük metnin satır kırılma özelliklerini belirler.

değerler ;

  • normal ; normal
  • break-word; kelimey ikır.
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

overflow-wrap: normal;
