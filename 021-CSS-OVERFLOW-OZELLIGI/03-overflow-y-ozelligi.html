overflow-y özelliği

Html elementinin y (dikey) ekseni baz alınarak boyutlandırılmış alanına sığmayacak kadar büyük içeriğin (resim,video,text gibi) taşma özelliklerini belirler.

değerler ;

  • visible ; görünür (varsayılan)
  • hidden ; gizli
  • scroll ; kaydırma ( kaysın kaymasın iki yönede scroll gösterir.)
  • auto ; otamatik değer alır (içerik ne tarafa taşıyorsa ona göre scroll gösterir.)
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır
