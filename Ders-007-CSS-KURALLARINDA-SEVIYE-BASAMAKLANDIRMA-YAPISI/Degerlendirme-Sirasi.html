CSS KURALLARINDA SEVİYE BASAMAKLANDIRMA YAPISI;

Css ile Html elemanlarına biçimlendirme yapıldığında, browser'lar web sayfalarını görüntülerken bir seviye basamaklandırma yapısı kurarlar. Bu sayede web sayfasının Css ile atanmış biçimlendirmesi oluşmuş olur. Bu işlem sırasında 4 temel birim üzerinden işlem gerçeleşir ve bir numaralandırma yapısı kurulur.

Değerlendirme sırası şu şekildedir.

  1. Satır içi CSS tanımı (1000)
  2. ID Tanımı (100)
  3. Class seçici ile Css tanımı (10)
  4. Element seçicisi ile Css tanımı(1)

ul li İkili element seçici 0002
