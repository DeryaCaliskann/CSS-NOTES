nth-last-child() pseudo class

Belirtilen html elementi baz alınarak,üst elementine göre kardeş element türüne bakılmaksızın son (çocuk)child durumundaki elementten başlayarak sıra numarasına göre stil tanımlar.Elemanları sırasını sayarken türüne bakmaksızın hepsini sayar.

odd: Tek haneye göre sıralama

even: Çift haneye göre sıralama

Örnek;

ul li:nth-last-child(3) {
color: red;
}

Alttan üçüncü elemana stil tanımlar.

ul li:nth-last-child(2n+1) {
background-color: #f1f1f1;
}

Altan başlayarak ikişer ikişer stil tanımlar

ul li:nth-last-child(odd) {
background-color: #f1f1f1;
}

Alttan başlayarak sıra numarası tek sayılı elemanlara işlem yapar.

ul li:nth-lats-child(even) {
background-color: #f1f1f1;
}

Alttan başlayarak sıra numarası çift sayılı elemanlara işlem yapar.

ul li:nth-last-child(n+4) {
font-weight: bold;
}

Alttan dört'ten başlayıp listenin başına kadar elemana işlem yapar.

ul li:nth-last-child(-n+4) {
font-style: italic;
}

Sondan dört'ten başlayıp listenin sonuna kadar elemana işlem yapar.

ul li:nth-child(1n+2):nth-last-child(1n+4) {
text-decoration: underline;
}

dört'ten başlayıp yukarı doğru ikinci satıra kadar işlem yapar.
