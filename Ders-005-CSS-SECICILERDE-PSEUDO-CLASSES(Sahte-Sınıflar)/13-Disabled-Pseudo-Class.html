disabled pseudo class

Belirtilen Html elementi etkin olamadığında stil tanımlar.Element ismi belirterek yada belirtmeden kullanabiliriz.

Örnek;

input:disabled {
color: red;
background-color:grey;
}
textarea:disabled {
color:red;
background-color:grey;
}
