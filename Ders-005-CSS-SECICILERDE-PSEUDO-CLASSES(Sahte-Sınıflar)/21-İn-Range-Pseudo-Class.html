IN-RANGE PSEUDO CLASS (SAHTE SINIFI)

Belirtilen html elementi bilgi girişi belirtilen aralıklar dahilinde bir değer olduğunda stil tanımlar. Belirtilen html elementin bilgi girişi belirtilen aralıklar dışında bir değer olduğunda stil normale döner.

Örnek ;

Adınız Soyadınız :

Yaşınız :

input[type="number"]:in-range {
color: red;
}
veya
input:in-range {
color: red;
}
Not: Her iki kullanım da doğru çalışır. Ama yukarıda ki kullanım şekli daha doğrudur.
