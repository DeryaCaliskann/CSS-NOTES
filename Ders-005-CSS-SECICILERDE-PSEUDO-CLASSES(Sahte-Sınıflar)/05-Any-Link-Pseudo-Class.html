ANY-LINK PSEUDO CLASS (SAHTE SINIFI)

Dökümanın içerisinde bulunan html'in link elementinin URL değeri boş değil ise stil tanımlar. any-link sınıfının her tarayıcıda doğru çalışması için prefix'lerle beraber kullanılır. any-link sahte sınıfının kullanımında URL değeri href olarak belirlendikten sonra içersine hiç bir şey yazılmasa dahi stil tanımlar.

GOOGLE
