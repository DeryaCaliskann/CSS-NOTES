list-style-image

Listedeki madde işaretçisinin resim özelliklerini belirler.

değerler ;

  • none; değer yok
  • url; dosya adresi
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

list-style-image:url(../img/kucukcicek.png);
