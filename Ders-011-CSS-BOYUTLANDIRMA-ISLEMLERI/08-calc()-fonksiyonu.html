calc () fonksiyonu

Değer hesaplama işlemleri için kullanılır. Risponsive sitelerde sıklıkla kullanılır. Bütün tarayıcılarda kusursuz çalışması için Prefixlerle beraber kullanılır. İşlemlerde sadece tam sayıları değil ondalıklı sayılarıda kullanabiliriz.

değerler ;

  1. + ;Toplama
  2. - ;Çıkarma
  3. * ;Çarpma
  4. / ;Bölme

Örnek-1 ;

calc(25% + 200px)

Örnek-2 ;

calc(50% - 200px)

Örnek-3 ;

calc(300px * 2)

Örnek-4 ;

calc(%50 / 2)
