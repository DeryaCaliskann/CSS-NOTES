HEİGHT(YÜKSEKLİK) ÖZELLİĞİNİN KULLANIMI

Html elementinin yüksekliğini belirtir.Yükseklik değeri genişlik değeri gibi değildir.İçeriğe göre değişir.Hiçbir zaman bir elementin yüksekliği kendi içeriğinden daha fazla olmaz.

DEĞERLER

  1. auto; Otomatik değer alır. Auto'da içeriğin yüksekliğinden fazla değer almaz.
  2. length;Yüksekliğin uzunluk değerini belirler. Ekran küçülürse scroll çıkar.İçteki eleman daha yüksek olursa taşma yapar.
  3. % ; Yüzde değeridir. Yüzdesel bir değer verdiğimizde içeriği baz alır.
  4. İnitial; Html elementi varsayılan biçimlendirme değerine döner.
  5. İnherit; Html elementi üst elementinin biçimlendirme değerini alır. Yani miras alır.

Örnek-1 ;

div: height;auto
içeriğe göre artar.

Örnek-2 ;

div: min-height;50px

Örnek-3 ;

div: height;100px (mavi) > p: height;50px (sarı)

Örnek-4;

div: height;50px (mavi) > div: height;100px (sarı)



Örnek-5 ;

div: height;100%

Örnek-6;

div: height;100px / div: height;initial (varsayılan)

Örnek-7;

div: height;100px (mavi) > p: height;inheri (sarı) (miras)
