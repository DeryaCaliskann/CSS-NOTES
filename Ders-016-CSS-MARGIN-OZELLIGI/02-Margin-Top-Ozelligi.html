MARGIN-TOP ÖZELLİĞİNİN KULLANIMI

Html elementinin üst kenar boşluğu (dış boşluk) özelliklerini belirler.

DEGERLER

  • length ; Uzunluk değeri
  • % ; yüzde değeri ;
  • auto ; otomatik değer alır. (kaydırma yapmaz)
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

margin-top: 100px;

Örnek-2 :

margin-top: 50px;

Örnek-3 :

margin-top: 10%;

Örnek-4 :

margin-top:auto;(tek başına kaydırma yapmaz.)

Örnek-5:

margin-top: initial;

Örnek-6:

margin-top: 50px;

margin-top: inherit;

margin-top: inherit;
