MARGIN ÖZELLİĞİNİN KULLANIMI

Html elementinin kenar boşluğu (dış boşluk) özelliklerini belirler. Bazı elementlerin varsayılan margin değerleri vardır. Örneğin p, body tagı gibi. Biçimlendirme yapmadan önce bunları sıfırlamalıyız (margin:0)

DEGERLER

  • length ; Uzunluk değeri
  • % ; yüzde değeri ;
  • auto ; otomatik değer alır
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

margin: 100px;

Örnek-2 :

margin: 100px 50px 75px 25px; /* üst, sağ, alt, sol) */

Örnek-3 :

margin: 10%;

Örnek-4 :

margin:0 auto; (sayfayı yatay ortalar)

Örnek-5:

margin: initial;

Örnek-6:

margin: 100px;

margin: inherit;
