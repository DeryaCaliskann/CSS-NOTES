CSS renk tanımları

Css,html elemementlerine biçimlendirme işlemi yaparken bir çok renk tanimlama çeşidini de aynı zamanda kullanabilmektedir.

Css'de renk tanımlama işlemleri 4 farklı grup ile sağlanmaktadır.

  • ana renk
  • rgb ve rgba
  • hsl ve hsla
  • hexadecimal (onaltılık)

ana renk

Html'de tanımlı olan bir ana renk, Css kodları içerisinde yazılarak kullanılabilir. Black, red, green, blue vs.

Örnek:

h1 {
color: red;
}
