CSS'DE OUTLINE-STYLE ÖZELLİĞİ

Html elementinin anahat stilini belirler.

DEĞERLER

  • none ; değer yok
  • hidden ; gizli
  • solid ; düz çizgi
  • double ; çift çizgi
  • dashed ; keşikli çizgi
  • dotted ;noktalı çizgi
  • groove ; içe dönük gölgeli çizgi
  • ridge ; dışa dönük gölgeli çizgi
  • inset ; içe dönük derin çizgi
  • outset ; dışa dönük derin çizgi
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

Örnek-1

outline-style: none; (değeri yok eder)

Örnek-2

outline-style: hidden;(özellikler parçalı girilirse stili gizler)

Örnek-3

outline-style:solid;

Örnek-4

outline-style:double;

Örnek-5

Örnek-6

Örnek-7

outline-style:groove;

Örnek-8

Örnek-9

outline-style:inset;

Örnek-10

Örnek-11

outline-style:initial;

Örnek-12
