CSS'DE OUTLINE-WIDTH ÖZELLİĞİ

Html elementinin anahat /dış çizgi genişliğini belirler

DEĞERLER :

  • length ; uzunluk
  • thin ; ince
  • medium ; orta
  • thick ; kalın
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

Örnek-1

outline-width: 2px;

Örnek-2

outline-width:thin;

Örnek-3

outline-width: medium;

Örnek-4

outline-width: thick;

Örnek-5

outline-width: initial;

Örnek-6

outline-width: 2px;

outline-width:inherit;
