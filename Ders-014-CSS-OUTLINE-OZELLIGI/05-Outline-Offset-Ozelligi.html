CSS'DE OUTLINE-OFFSET ÖZELLİĞİ

Html elementi içi anahat ile kenarlık arasındaki mesafeyi belirler. Negatif deeğer de alabilir.

DEĞERLER

  • length ; uzunluk
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

Örnek-1

outline-offset: 5px;

Örnek-2

outline-offset: 5px;

Örnek-3

outline-offset: -5px;

Örnek-4

outline-offset: initial;

Örnek-5

div: outline-offset: 5px;

p: outline-offset: inherit;
