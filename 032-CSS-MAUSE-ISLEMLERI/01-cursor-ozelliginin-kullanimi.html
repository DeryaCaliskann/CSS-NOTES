cursor özelliği;

Mause (fare) işaretçisini stilini belirler.

Not: Bazı komutlar aynı işaretleri verir. Bunun nedeni işletim sistemlerine göre değişiklik göstermelerinden dolayıdır.

değerler ;

  1. auto ; otomatik
  2. default ; mause imleci ok şeklinde görünür.
  3. none ; değer yok. imleç görünmez.
  4. pointer ; el görüntüsü.
  5. text ; imleç görüntüsü.
  6. vertical-text ; yatık imleç.
  7. crosshair ; artı şekli.
  8. cell ; dolgulu artı.
  9. help ; soru işareti.
  10. alias ;küçük ok görüntüsü ekler.
  11. copy ;küçük artı değer ekler.
  12. progress ; dönen daire (yükleniyor)
  13. wait ; dönen daire oksuz.
  14. zoom-in ;büyüteç, yaklaş (+)
  15. zoom-out ; büyüteç, uzaklaş (-)
  16. grab ;açık el şekli sürükle bırak (prefix tanımlanır)
  17. grabbing ; el kapalı
  18. move ; sürükleme işareti
  19. no-drop ; dur işareti
  20. not-allowed ; dur işareti
  21. all-scroll ; sürükleme işareti
  22. row-resize ; satır boyutlandırma imleci
  23. col-resize ; sütun boyutlandırma imleci
  24. nwse-resize ; sol üstten sağ alta doğru çift taraflı ok
  25. se-resize ; sol üstten sağ alta doğru çift taraflı ok ok
  26. nw-resize ; sol üstten sağ alta doğru çift taraflı ok
  27. nesw-resize ; sağ üstten sol alta doğru çift taraflı ok
  28. sw-resize ; sağ üstten sol alta doğru çift taraflı ok
  29. ne-resize ; sağ üstten sol alta doğru çift taraflı ok
  30. ns-resize ; dik çift taraflı ok
  31. n-resize ; dik çift taraflı ok
  32. es-resize ; dik çift taraflı ok
  33. ew-resize ; yatık çift taraflı ok
  34. e-resize ; yatık çift taraflı ok
  35. e-resize ; yatık çift taraflı ok
  36. w-resize ; yatık çift taraflı ok
  37. url ; "cur"uzantılı resim koyar.
  38. initial ; varsayılan değere döner.
  39. inherit ;üst elementini miras alır.
