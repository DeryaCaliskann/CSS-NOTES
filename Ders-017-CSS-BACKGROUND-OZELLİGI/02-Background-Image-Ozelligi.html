BACKGROUND-IMAGE ÖZELLİĞİ

html elementinin arka plan resmini belirler

DEĞERLER ;

  • none ; değer yok (varsayılan değer)
  • image ; resim
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

background-image: url(../img/desen2.jpg);

Örnek-2 ;

background-image: none;

Örnek-3 ;

background-image: initial;(varsayılan)

Örnek-4;

background-image: url(../img/desen-3.jpeg);

background-image: inherit; (miras)
