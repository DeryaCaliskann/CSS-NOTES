BACKGROUND-ATTACHMENT ÖZELLİĞİ

Html elementine atanmış olan arka plan resminin sayfa hareketlerine göre davranış özelliklerini belirler.

DEĞERLER ;

  • scroll ; kaydırma (elementin scroll'u kaydığında sabit kalır)
  • fixed ; sabit (ister sayfanın ister elementin scroll'u kaysın her zaman sabit kalır.)
  • local ; yerel (elementin scroll'u kaydığında içerikle beraber resim de kayar.)
  • initial ; html elementi varsayılan biçimlendirme değerine döner. Varsayılan değer scroll'dur.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.
