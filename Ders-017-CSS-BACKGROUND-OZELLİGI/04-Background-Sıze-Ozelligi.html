BACKGROUND-SIZE ÖZELLİĞİ

html elementine atanmış olan arka plan resminin boyutunu belirler.Kendi içinde çift yada tek parametreli kullanım yöntemleri var.

DEĞERLER ;

  • auto ;otomatik değer alır.Varsayılan değerdir.Sol üst köşeden resmi ekrana sığdığı kadar ekler.Başka bir müdahalede bulunmaz.
  • length ;uzunluk değeri. Tek parametre verildiğinde yatay eksende yani genişlik değerini vermiş oluyoruz. Genişlik değerini uyarlarken yüksekliğide otomatik olarak alır. Orantılı bir boyutlandırma işlemi yapar. İkiparametre aldığında ilki genişlik ikincisi yüksekliktir.
  • % ;yüzde değeri. İki parametreli olarak kullanılabilir. İlk parametre genişlik ikinci parametre yüksekliktir. Genelde tek parametre kullanılır. Tek parametre otomatik ayarlıyor.
  • cover ;kapak. Yatayda kayıp vermez dikeyde resim nereye kadar geliyorsa serer. Yani enini hiçbir zaman bozmaz boyunu tarayıcının yüksekliğine göre otomatik ayarlar.
  • contain ;içerik demek. Ne kadar içerik olursa resim kendini ona göre boyutlandırır.
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

background-size: 500px 200px;

Örnek-2 ;

background-size: 250px

Örnek-3;

background-size: 100%;

Örnek-4 ;

background-size: 100% 100%;

Örnek-5 ;

background-size: cover;

Örnek-6 ;

background-size: contain;

Örnek-7 ;

background-image: initial;(varsayılan)

Örnek-8;

background-size: 100%;

background-size:inherit
