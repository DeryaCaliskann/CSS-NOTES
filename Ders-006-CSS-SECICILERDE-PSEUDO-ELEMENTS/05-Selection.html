5-selection pseudo element

Belirtilen Html elementinin kullanıcı tarafından seçilen kısmına stil tanımlar.Yani mause ile sağ tıklayıp seçtiğimiz kısım. Selection sahte sınıfının bütün tarayıcılarda doğru çalışması için prefix'lerle beraber kullanmalıyız.

Örnek ;

p::-webkit-selection {
background-color: red;
color: white;
}

p::-moz-selection {
background-color: red;
color: white;
}

p::-o-selection {
background-color: red;
color: white;
}

p::-ms-selection {
background-color: red;
color: white;
}

p::selection {
background-color: red;
color: white;
}
