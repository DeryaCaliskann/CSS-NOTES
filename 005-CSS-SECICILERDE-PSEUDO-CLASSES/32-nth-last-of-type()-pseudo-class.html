nth-last-of-type() pseudo class

Belirtilen html elementi baz alınarak,üst elementine göre kardeş element türleride eşleşen son (çocuk)child durumundaki elementten başlayarak sıra numarasına göre stil tanımlar. Elemanları sırasını sayarken sadece belirttiğimiz türde olanları sayar.

odd: Tek haneye göre sıralama

even: Çift haneye göre sıralama


ul li:nth-last-of-type(3) {
color: red;
}

Alttan üçüncü elemana stil tanımlar.

ul li:nth-last-of-type(2n+1) {
background-color: #f1f1f1;
}

Altan başlayarak ikişer ikişer stil tanımlar

ul li:nth-last-of-type(odd) {
background-color: #f1f1f1;
}

Alttan başlayarak sıra numarası tek sayılı elemanlara işlem yapar.

ul li:nth-lats-of-type(even) {
background-color: #f1f1f1;
}

Alttan başlayarak sıra numarası çift sayılı elemanlara işlem yapar.

ul li:nth-last-of-type(n+4) {
font-weight: bold;
}

Alttan dört'ten başlayıp listenin başına kadar elemana işlem yapar.

ul li:nth-last-of-type(-n+4) {
font-style: italic;
}

Sondan dört'ten başlayıp listenin sonuna kadar elemana işlem yapar.

ul li:nth-of-type(1n+2):nth-last-of-type(1n+4) {
text-decoration: underline;
}

dört'ten başlayıp yukarı doğru ikinci satıra kadar işlem yapar.
