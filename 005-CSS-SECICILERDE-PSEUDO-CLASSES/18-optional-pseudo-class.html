optional pseudo class

Elemente bilgi girişi opsiyonel olduğunda stil tanımlar.Yani required sahte sınıfının tam tersine bilgi girişi zorunlu olmayan html elementine stil tanımlar.

Örnek ;

input:optional {
border: 2px solid green;
}

input:required {
border: 2px solid red;
}
