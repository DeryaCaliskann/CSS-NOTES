invalid pseudo class

Belirtilen html elementin bilgi girişi geçersiz bir değer olduğunda stil tanımlar. Belirtilen html elementi bilgigirişi geçerli bir değer olduğunda stil normale döner. Örneğin e-mail formatının dışında bir giriş yaparsak stil tanımlar. Geçerli bir e-mail formatı girdiğimizde stil normale döner.Valid sahte sınıfını tam tersiişlem yapar.

Örnek -1 :

input:invalid {
background-color: red;
}

input:valid {
background-color: green;
}
