out-of-range pseudo class

Belirtilen html elementin bilgi girişi belirtilen aralıklar dışında bir değer olduğunda stil tanımlar. Belirtilen aralıklar dahilinde olursa stil normale döner. in-range sahte sınıfının tam tersi işlem yapar.

Örnek ;

input[type="number"]:out-of-range {
color: red;
}
veya
input:out-of-range {
color: red;
}
Not: Her iki kullanım da doğru çalışır. Ama yukarıda ki kullanım şekli daha doğrudur.
