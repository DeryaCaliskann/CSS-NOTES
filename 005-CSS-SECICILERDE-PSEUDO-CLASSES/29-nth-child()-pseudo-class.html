nth-child() pseudo class

Belirtilen html elementi baz alınarak,üst elementine göre kardeş element türüne bakılmaksızın ilk (çocuk)child durumundaki elementten başlayarak sıra numarasına göre stil tanımlar.Elemanları sırasını sayarken türüne bakmaksızın hepsini sayar.

odd: Tek haneye göre sıralama

even: Çift haneye göre sıralama

Örnek;

ul li:nth-child(3) {
color: red;
}

Üçüncü elemana stil tanımlar.

ul li:nth-child(2n+1) {
background-color: #f1f1f1;
}

İlk elemandan başlayarak ikişer ikişer stil tanımlar

ul li:nth-child(odd) {
background-color: #f1f1f1;
}

Sıra numarası tek sayılı elemanlara işlem yapar.

ul li:nth-child(even) {
background-color: #f1f1f1;
}

Sıra numarası çift sayılı elemanlara işlem yapar.

ul li:nth-child(n+4) {
font-weight: bold;
}

Dört'ten başlayıp listenin sonuna kadar elemana işlem yapar.

ul li:nth-child(-n+4) {
font-style: italic;
}

Dört'ten başlayıp listenin başına kadar elemana işlem yapar.

ul li:nth-child(1n+2):nth-last-child(1n+4) {
text-decoration: underline;
}

dört'ten başlayıp yukarı doğru ikinci satıra kadar işlem yapar.
