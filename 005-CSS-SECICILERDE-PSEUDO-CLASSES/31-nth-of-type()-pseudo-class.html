nth-of-type() pseudo class

Belirtilen html elementi baz alınarak,üst elementine göre kardeş element türleri de eşleşen ilk (çocuk)child durumundaki elementten başlayarak sıra numarasına göre stil tanımlar. Aynı türde kardeş elemanlar olduğu müddetçe nth-child () sahte sınıfıyla aynı çalışır. Elemanları sırasını sayarken sadece belirttiğimiz türde olanları sayar.

odd: Tek haneye göre sıralama

even: Çift haneye göre sıralama


ul li:nth-of-type(3) {
color: red;
}

Üçüncü elemana stil tanımlar.

ul li:nth-of-type(2n+1) {
background-color: #f1f1f1;
}

İlk elemandan başlayarak ikişer ikişer stil tanımlar

ul li:nth-of-type(odd) {
background-color: #f1f1f1;
}

Sıra numarası tek sayılı elemanlara işlem yapar.

ul li:nth-of-type(even) {
background-color: #f1f1f1;
}

Sıra numarası çift sayılı elemanlara işlem yapar.

ul li:nth-of-type(n+4) {
font-weight: bold;
}

Dört'ten başlayıp listenin sonuna kadar elemana işlem yapar.

ul li:nth-of-type(-n+4) {
font-style: italic;
}

Dört'ten başlayıp listenin başına kadar elemana işlem yapar.

ul li:nth-of-type(1n+2):nth-last-of-type(1n+4) {
text-decoration: underline;
}

dört'ten başlayıp yukarı doğru ikinci satıra kadar işlem yapar.
