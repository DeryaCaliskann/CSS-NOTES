empty pseudp class

Belirtilen Html elementinin içeriği tamamen boş ise stil tanımlar. Sahte sınıfımızın görünebilir duruma gelmesi için boyut vermemiz gerekir. Element belirterek veya belirtmeden kullanabiliriz.

Örnek;

div:empty {
background-color: blue;
border: 5px solid yellow;
margin: 2px 0;
width: 50%;
height: 50px;
}
