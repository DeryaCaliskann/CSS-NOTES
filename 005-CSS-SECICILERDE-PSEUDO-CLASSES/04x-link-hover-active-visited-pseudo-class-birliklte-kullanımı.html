link,hover,active,visited pseudo class'ların birlikte kullanımı

Bu sahte sınıflar genellikle beraber kullanılır. Bu dört sınıf genelde link taglarında kullanılır. Çünki link tagları kendine özel bir biçimlendirmeye sahiptir. Bu sınıfları html taglarının özellik etiketleriyle birliktede kullanabiliriz.

Örnek ;

a[lang=tr-TR]:link,a[lang=tr-TR]:visited,a[lang=tr-TR]:hover,a[lang=tr-TR]:active {
color: black;
text-decoration: none;
font-weight: bold;
}
