valid pseudo class

Belirtilen html elementin bilgi girişi geçerli bir değer olduğunda stil tanımlar. Belirtilen html elementi bilgi girişi geçersiz bir değer olduğunda stil normale döner. Örneğin e-mail input alanına e-mail formatının dışında bir giriş yaptığımızda stil normale döner.

Örnek -1 :

input:valid {
background-color:green;
}
