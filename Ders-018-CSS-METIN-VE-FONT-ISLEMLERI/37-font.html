font özelliği

Metnin tüm yazı tipi özelliklerini belirler.

değerler ;

  • font-style; metnin yazı tipi sitili
  • font-variant; metnin küçük harf içerikleri için büyük harf gösterilme özelliği.
  • font-weight; metnin yazı tipi kalınlığı.
  • font-size; metnin yazı tipi boyutu.
  • line-height; metin bloğunda bulunan satırların yüksekliği.
  • font-family; metnin yazı tipi ailesi.
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.
not: font ailesini ve font boyutunu muhakkak belirtmemiz gerekiyor.
