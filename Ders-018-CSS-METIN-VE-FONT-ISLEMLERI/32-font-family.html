font-family

Metnin yazı tipi ailesini belirler

değerler ;

  • family-name; yazı tipi ailesinin adı
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.

Font ailelerinin türleri ;

  • family-name ; yazı tipi ailesinin adı
  • generic-name ; genel yazı tipi ailesin adı
    • serif; tırnaklı
    • sans-serif; tırnaksız
    • cursive; el yazısı
    • fantasy; fantazi
    • monospace; tek aralıklı
