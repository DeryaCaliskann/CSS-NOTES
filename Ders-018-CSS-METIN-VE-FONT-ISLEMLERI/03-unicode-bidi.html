unicode-bidi

Metnin direction özelliği ile belirtilmiş akış yönü özelliğinin döndürme durumunu belirler.

değerler ;

  • normal; normal değer.
  • embed; gömülü demek.Tek element varsa normalden farkı olmaz. İç elementler varsa bunlarada değer tanımlanmışsa embed değeriyle metnin yönünü döndüre biliyoruz. (örnek-4 deki gibi)
  • bidi-override; geçersiz kılma.
  • initial; html elementi varsayılan biçimlendirme değerine döner.
  • inherit; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

unicode-bidi: normal;
Lorem ipsum dolor sit amet consectetur adipisicing elit. Reiciendis nesciunt voluptatibus sapiente alias veritatis. Dolor dignissimos id delectus, omnis amet placeat explicabo exercitationem.
