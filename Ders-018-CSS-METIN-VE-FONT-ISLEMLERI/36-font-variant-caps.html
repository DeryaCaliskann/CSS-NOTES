font-variant-caps

Metnin küçük harf içerikleri için büyük harf gösterilme içeriklerini belirler.

değerler ;

  • normal; normal
  • small-caps; küçük harfli metinler, küçük harf boyutu korunarak, büyük harf olarak dönüştürülür.
  • all-small-caps; tüm metinler, küçük harf boyutu korunarak, büyük harf olarak dönüştürülür.
  • petite-caps; küçük harfli metinler, küçük harf boyutu korunarak, büyük harf olarak dönüştürülür. (yazı tipi ailesi küçük harf büyüklüğünü desteklemiyorsa small-caps yerine kullanılır.)
  • all-petite-caps; tüm metinler, küçük harf boyutu korunarak, büyük harf olarak dönüştürülür. (yazı tipi ailesi küçük harf büyüklüğünü desteklemiyorsa all-small-caps yerine kullanılır.
  • unicase; Büyük harfli metinler küçük harf boyutu korunarak büyük harf olarak dönüştürülür.
  • İnitial; html elementi varsayılan biçimlendirme değerine döner.
  • İnherit; html elementi üst elementinin biçimlendirme değerini alır.
