column-rule özelliği

Html elementinin çok sütunlu düzeni için bölünen sütununların aralarındaki dikey ayraç özelliklerini belirler. Prefix'lerle beraber kullanılır.

parametreler ;

  • column-rule-width
  • column-rule-style
  • column-rule-color

değerler ;

  • rule value ; kural değeri
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.
