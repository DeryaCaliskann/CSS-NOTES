column-rule-color özelliği

Html elementinin çok sütunlu düzeni için bölünen sütunların aralarındaki dikey ayraç rengini belirler. Prefix'lerle beraber kullanılır.

değerler ;

  • color ; renk
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

column-rule-color: blue;
