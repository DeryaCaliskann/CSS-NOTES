break-after özelliği

Html elementinin çok sütunlu düzeni için bölünen sütununların arasındaki belirtilecek olan elementin öncesinde bölünme davranış özelliklerini belirler. Prefix'lerle beraber kullanılır.

değerler ;

  • auto ; otomatik değer alır (varsayılan)
  • column ; sütunu böler
  • initial ; html elementi varsayılan biçimlendirme değerine döner.

Örnek-1

break-after: auto;
