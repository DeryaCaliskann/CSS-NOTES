columns özelliği

Html elementinin çok sütunlu düzeni için bölünmesi gereken sütunların minimum genişlik ve maksimum sütun sayısı özelliklerini belirler.

paramtreler;

  • column-width
  • column-count

değerler ;

  • auto ; otamatik değer alır (varsayılan değerdir.)
  • column-width ; sütun genişlik değereri.
  • column-count ; sütun adedi genişliği.
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.
