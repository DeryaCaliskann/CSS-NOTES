column-span özelliği

Html elementinin çok sütunlu düzeni için bölünen sütunların içerisindeki bir elementin sütun veya sütunlara yayılması durumunu belirler. Prefix'lerle beraber kullanılır.

değerler ;

  • none ; değer yok (varsayılan)
  • all ; tümü
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

column-span: none;
