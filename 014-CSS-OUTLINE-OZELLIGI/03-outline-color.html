CSS'de outline-color özelliği

Html elementinin anahat rengini belirler.

değerler ;

  • color ; renk
  • invert ;ters çevir
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

Örnek-1

outline-color: blue;

Örnek-2

outline-color:invert;(tutarsız tarayıcılarda çalışır.)

Örnek-3

outline-color:initial;

Örnek-4

outline-color: blue;

outline-color:inherit;
