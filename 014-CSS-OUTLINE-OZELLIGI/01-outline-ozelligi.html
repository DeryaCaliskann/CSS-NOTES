CSS'de outline özelliği

Html elementinin anahat /dış çizgi özelliliklerini belirler. Border özelliğine benzer ama aynı değildir. border sayfa akışına etki eder outline etmez. Border her kenar için farklı olabilir ama outline bütün kenarlar için aynı verilebilir. Outline bir kenarlık değildir. Kenarlığın dışındaki anahat çizgisidir. Her kenar için ayrı ayrı değerler verilemez.

parametreler ;

1 - length ;

  • length ; uzunluk
  • thin ; ince
  • medium ; orta
  • thick ; kalın
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.

2 - style

  • none ; değer yok
  • hidden ; gizli
  • solid ; düz çizgi
  • double ; çift çizgi
  • dashed ; keşikli çizgi
  • dotted ;noktalı çizgi
  • groove ; içe dönük gölgeli çizgi
  • ridge ; dışa dönük gölgeli çizgi
  • inset ; içe dönük derin çizgi
  • outset ; dışa dönük derin çizgi

3 - color

  • color ; renk
  • invert ;ters çevir
