PADDİG-TOP ÖZELLİĞİNİN KULLANIMI

Html elementinin üst dolgu (iç boşluk) özelliklerini belirler.

DEGERLER

  • length ; uzunluk değeri ;
  • % ;yüzde değeri ;
  • initial; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ;html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

padding-top: 25px;

Örnek-2:

padding-top:5%;

Örnek-3:

padding-top:initial;

Örnek-4:

padding-top:inherit
