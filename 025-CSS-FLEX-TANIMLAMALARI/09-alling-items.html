align- items özelliği

Esnek kutu içerisinde bulunan elementlerin hizalama özelliklerini belirler.

değerler ;

  • stretch ; uzat / esne (varsayılan)
  • center ; orta
  • flex-start ; esnek başlangıç
  • flex-end ; esnek bitiş
  • baseline ; başlangıç
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
