align-self özelliği

Esnek kutu içerisinde bulunan, seçilen element veya elementlerin hizalama özelliklerini belirler.

değerler ;

  • auto; otomatik değer alır.(varsayılan)
  • stretch ; uzat / esne
  • center ; orta
  • flex-start ; esnek başlangıç
  • flex-end ; esnek bitiş
  • baseline ; başlangıç
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
