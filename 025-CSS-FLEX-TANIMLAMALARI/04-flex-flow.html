flex-flow özelliği

Esnek kutuların dizilim yönü ve satır kırılma özelliklerini belirler.

parametreler ;

  • flex-direction
    • row ; satır (varsayılan)
    • row-reverse ; ters satır
    • column ; sütun
    • column-reverse ; ters sütun
  • flex-wrap
    • nowrap ; kırılma yok (varsayılan)
    • wrap ; kırılma
    • wrap-reverse ; ters kırılma
  • initial ;html elementi varsaylan biçimlendirme değerine döner.
