justify-content özelliği

Esnek kutu içerisinde bulunan elementler tüm alanı kaplamadığında elementlerin arasındaki hizalama özelliklerini belirler.

değerler ;

  • flex-start ; esnek başlangıç (varsayılan)
  • flex-end ; esnek bitiş
  • center ; orta
  • space-between ; boşluk arasında
  • space-around ; boşluk etrafında
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
