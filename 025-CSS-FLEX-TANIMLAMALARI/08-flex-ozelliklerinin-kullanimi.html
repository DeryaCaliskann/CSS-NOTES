flex özelliği

Esnek kutu içerindeki element veya elementlerin diğer elementlere göre büyüklük oranını yada küçüklük /büzülme oranını yada başlangıç uzunluğu değerini belirler. Bu özelliği genellikle eşit bölmek için kullanırız. Üç kombinasyon aynı anda çalışır.

değerler ;

  • flex-grow ; Esnek kutu içerindeki element veya elementlerin diğer elementlere göre büyüklük oranı. Varsayılan 0'dır.
  • flex-shrink ; Esnek kutu içerindeki element veya elementlerin diğer elementlere göre küçüklük / büzülme oranı. Varsayılan 1'dir
  • flex-basis ; Esnek kutu içerindeki element veya elementlerin başlangıç uzunluğu. Varsayılan auto.
  • auto ; otamatik değer alır
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
