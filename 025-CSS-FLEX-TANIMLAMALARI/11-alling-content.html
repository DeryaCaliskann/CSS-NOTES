align-content özelliği

Esnek kutu içerisinde bulunan elementler tüm alanı kullanmadığında ve satır kırılma olduğunda elementlerin hizalama özelliklerini belirler.

değerler ;

  • stretch ; uzat / esne (varsayılan)
  • center ; orta
  • flex-start ; esnek başlangıç
  • flex-end ; esnek bitiş
  • space-between ; boşluk arasında
  • space-around ; boşluk etrafında
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
