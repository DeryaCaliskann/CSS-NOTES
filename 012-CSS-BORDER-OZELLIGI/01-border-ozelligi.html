border özelliği

Html elementinin kenarlık, çerçeve özelliliklerini belirler.

parametreler;

Length, Style , Color parametrelerini alır.

1 - length

değerler ;

  • length ; uzunluk
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.
  • thin ; ince
  • medium ; orta
  • thick ; kalın

2 - style

değerler ;

  • none ; değer yok
  • hidden ; gizli
  • solid ; düz çizgi
  • double ; çift çizgi
  • dashed ; keşikli çizgi
  • dotted ;noktalı çizgi
  • groove ; içe dönük gölgeli çizgi
  • ridge ; dışa dönük gölgeli çizgi
  • inset ; içe dönük derin çizgi
  • outset; dışa dönük derin çizgi

3 - color

  • Kenarlıkları renklerini belirler.
Kenarlıkları renklerini belirler.

border: 1px solid blue
