border-radius özelliği

Html elementinin kenarlık köşelerini yuvarlaklığını belirler. Bütün tarayıcılarda sorunsuz çalışması için prefix'leri tanımlamalıyız. Her köşesine ayrı parametre verebiliriz. Diğer özellikler gibi saat yönünde çalışır.

değerler

  • lenght; uzunluk
  • initial ; html elementi varsayılan değerine döner.
  • inherit ; html elementi üst elementinin değerini alır.
  • % ; yüzde

Örnek-1;

border-radius: 20px;

Örnek-2;

border-radius: 20%;

Örnek-3;

border-radius: initial;(varsayılan)

Örnek-4;

border-radius: inherit;(miras)

Örnek-5;

border-radius:10px 20px 30px 40px;(üst,sağ,alt,sol)

Örnek-6;

border-radius:10px 20px 30px;(üst, sağ-sol,alt)

Örnek-7;

border-radius:10px 20px; (üst-alt, sağ-sol)
