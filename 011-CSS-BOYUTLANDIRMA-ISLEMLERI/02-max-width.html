max-width özelliği

Html elementinin maksimum genişliğini belirler.

değerler ;

  1. none; Değer yok. Üst elemanının kapladığı alan kadar yer kaplar. Maksimum genişlik değeri verilmemiş anlamına gelir.Kapsayabileceği maksimum genişliği kapsar.
  2. length;Uzunluk değeri. Sayfa küçülünce küçülür. Maksimum verilen uzunluk değerine kadar göster demek.
  3. % ; Yüzde değeridir.
  4. İnitial(varsayılan); Html elementi varsayılan biçimlendirme değerine döner.
  5. İnherit(miras); Html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

div: max-width;none

Örnek-2 ;

div: max-width;750px

Örnek-3

div: width;750px (kırmızı) > p: max-width;500px (sarı)

Örnek-4

div: width;500px (kırmızı)> p: max-width;750px (sarı)

Örnek-5

div: width;75% (kırmızı) > p: max-width;50% (sarı)

Örnek-6

div: width;50% (kırmızı) > p: max-width;75% (sarı)

Örnek-7 ;

div: max-width;initial(varsayılan)

Örnek-8

div: max-width;75% (kırmızı) > p: max-width;inherit(miras) (sarı)
