width (genişlik) özelliği

Html elementinin genişliğini belirtir.

değerler ;

  1. auto; Otomatik değer alır.Üst elementin genişliği ne kadarsa o kadar değer alır.Ekran küçülürse küçülür büyürse büyür.
  2. length;Genişliğin uzunluk değerini belirler. Length değeri atanırsa üst elementine göre yada üst elementi baz aldırmadan da değer verebiliriz.Elementleri iç içe kullandığımızda alttaki ölçeklendirme dğeri üsttekini geçebilir.Dışarı taşmasın istiyorsak overflov özelliğini kullanabiliriz.
  3. % ; Yüzde değeridir. İlgili üst elementi baz alır ve boyutlandırma yapar. Aynı auto'da olduğu gibi üst elementinin bir genişlik değeri yoksa bady'nin yani tarayıcının genişlik değerini baz alır.Auto değerinde olduğu gibi ekran büyürse büyür küçülürse küçülür.
  4. İnitial; Html elementi varsayılan biçimlendirme değerine döner.
  5. İnherit; Html elementi üst elementinin biçimlendirme değerini alır. Yani miras alır.

Örnek-1 ;

div: width;auto

Örnek-2 ;

div: width;750px

Örnek-3 ;

div: width;50%

Örnek-4 ;

div: width;500px / div: width;initial(varsayılan)

Örnek-5 ;

div: width;75% (kırmızı) > p: width;inherit(miras) (sarı)
