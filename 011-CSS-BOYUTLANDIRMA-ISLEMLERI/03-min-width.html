min-width özelliği

Html elementinin minimum genişliğini belirler.

değerler ;

  1. length;Uzunluk değeri. Minimum belirtilen değere kadar küçülür daha fazla küçülmez. Daha fazla küçültürsek altta scrool çubuğu çıkar. içerisinde olduğu elementin dışına taşabilir.
  2. % ; Yüzde değeridir.
  3. İnitial(varsayılan); Html elementi varsayılan biçimlendirme değerine döner.
  4. İnherit(miras); Html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

div: min-width;750px

Örnek-2 ;

div: width;80% (mavi) > p:min-width;500px (sarı)

Örnek-3;

div: width;500px (mavi) > div:min-width;750px (sarı)

Örnek-4 ;

div: width;80% (mavi) > div: min-width;50% (sarı)

Örnek-5 ;

div: min-width;750px / div: min-width;initial(varsayılan)

Örnek-6;

div: min-width;500px (mavi) > p:min-width;inherit(miras) (sarı)
