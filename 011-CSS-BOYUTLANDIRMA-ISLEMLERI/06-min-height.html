min-height özelliği

Html elementinin minimum yüksekliğini belirler. Yukarı çıkabilirsin ama azalamazsın demek oluyor.

değerler ;

  1. length;Uzunluk değeri.
  2. % ; Yüzde değeridir. Minimum yükseklik değeridir. Bunun altına düşemezsin ama yükselebilirsin demek.
  3. İnitial(varsayılan); Html elementi varsayılan biçimlendirme değerine döner.
  4. İnherit(miras); Html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

div: min-height;10px

Örnek-2 ;

div: min-height;50px

Örnek-3 ;

div: min-height;10px (mavi) > p: min-height;20px (sarı)

Örnek-4 ;

div: min-height;50%

Örnek-5;

div: min-height;50px / div: min-height;initial (varsayılan)

Örnek-6;

div: min-height;10px (mavi) > p: min-height;inherit(miras)(sarı)
