BORDER-IMAGE-OUTSET ÖZELLİĞİNİN KULLANIMI

Html elementinin kenarlık sınırından taşama özelliklerini belirler.

DEĞERLER:

  • length ; uzunluk
  • initial ; html elementi varsayılan değere döner.
  • inherit ;html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

border-image-outset:10px;

Örnek-2 :

border-image-outset: 10px 200px 50px 20px;



Örnek-3 :

border-image-outset: 20px 10px 30px;



Örnek-4 :

border-image-outset: 20px 10px;


Örnek-5 :

border-image-repeat:initial;


Örnek-6 :

border-image-outset: 3px;

border-image-outset: inherit;
