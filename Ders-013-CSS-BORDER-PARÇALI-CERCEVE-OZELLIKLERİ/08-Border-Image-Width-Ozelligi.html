BORDER-IMAGE-WIDTH ÖZELLİĞİNİN KULLANIMI

Html elementinin kenarlık resminin genişliğini belirler.

DEĞERLER:

  • auto; otomatik değer alır. (varsayılan değerdir)
  • length ; uzunluk değeri
  • % ; yüzde değeri
  • initial ; html elementi varsayılan değere döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

border-image-width: auto;

Örnek-2 :

border-image-width: 15px;

Örnek-3 :

border-image-width: 15px; 30px;

Örnek-4 :

border-image-width: 30px 30px 10px;

Örnek-5 :

border-image-width: 30px 30px 15px 25px;

Örnek-6 :

border-image-width: 20%;

Örnek-7 :

border-image-width:initial ;

Örnek-8 :

border-image-width:10px;

border-image-width: initial;
