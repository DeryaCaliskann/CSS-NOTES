CLASS (SINIF)SEÇİCİ

Elementin class değeri ile eşleşen html elemanını seçer.Class seçici kullanırken nokta işareti kullanılır. Bir class ismi belirleyip istediğimiz kadar elemana aynı class ismini kullanabiliriz.Oluşturduğumuz class isimlerini belli tag'lara uygulamak istiyorsak, önlerine tag isimlerini yazmalıyız.Bir Html etiketine birden fazla sınıf da atayabiliriz class adından sonra bir boşluk bırakıp ikinci class adınıda yazabiliriz. istediğimiz kadar sınıf adı yazabiliriz.
