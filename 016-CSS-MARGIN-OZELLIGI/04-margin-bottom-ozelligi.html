margin-bottom

Html elementinin alt kenar boşluğu (dış boşluk) özelliklerini belirler.

değerler ;

  • length ; Uzunluk değeri
  • % ; yüzde değeri ;
  • auto ; otomatik değer alır. (tek başına kaydırma yapmaz)
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

margin-bottom: 100px;

Örnek-2 :

margin-bottom:10%;

Örnek-3 :

margin-bottom: initial;

Örnek-4 :

margin-bottom;100px

margin-bottom;inherit

Örnek-5 :

margin-bottom: auto;
