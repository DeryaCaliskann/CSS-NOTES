margin-right

Html elementinin sağ kenar boşluğu (dış boşluk) özelliklerini belirler.

değerler ;

  • length ; Uzunluk değeri
  • % ; yüzde değeri ;
  • auto ; otomatik değer alır. (tek başına kaydırma yapmaz)
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

margin-right: 50px; ikinci span etiketi

Örnek-2 :

margin-right: 10%; ikinci span etiketi

Örnek-3:

margin-right:initial ikinci span etiketi

Örnek-4:

margin-right:inherit margin-right:inherit margin-right:inherit

Örnek-5 :

margin-right:auto ikinci span etiketi
