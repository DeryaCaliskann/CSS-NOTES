margin-left

Html elementinin sol kenar boşluğu (dış boşluk) özelliklerini belirler.

değerler ;

  • length ; Uzunluk değeri
  • % ; yüzde değeri ;
  • auto ; otomatik değer alır. (tek başına kaydırma yapmaz)
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 :

margin-left: 100px;

Örnek-2 :

margin-left:10%;

Örnek-3:

margin-left: initial;

Örnek-4:

margin-left;100px
margin-left: inherit;

Örnek-5:


margin-left: auto;
