border-spacing

Html'in table (tablo) elementinde bitişik hücrelerin arasındaki mesafeyi belirler. ilk yazdığımız parametre x (yatay), ikinci parametre y (dikey) ekseninde çalışır. border-collapse: collapse değeri verilecek olur ise border-spacing özelliği devredışı kalır.

değerler ;

  • length; uzunluk
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

border-spacing: 20px;
