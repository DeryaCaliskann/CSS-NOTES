empty-cells

Html'in table (tablo) elementinde bulunan ve içeriği boş hücrelerin görüntülenme özelliklerini belirler.

değerler ;

  • show ; göster. (varsayılan)
  • hide ; gizle.
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

empty-cells: show;
Pazartesi Salı Çarşamba Perşembe Cuma Cumartesi Pazar
Türkçe Matematik Fen Coğrafya
Türkçe Tarih Beden Müzik Kimya
Müzik Fizik Din Türkçe Türkçe Beden

Örnek-2 ;

empty-cells: hide;
Pazartesi Salı Çarşamba Perşembe Cuma Cumartesi Pazar
Türkçe Matematik Fen Coğrafya
Türkçe Tarih Beden Müzik Kimya
Müzik Fizik Din Türkçe Türkçe Beden
Not: Genellikle tabloya kenarlık vermeden kullanılır. Oyunlarda sık kullanılır.
