table-layout

Html'in table (tablo) elemtinde bulunan sütünların genişlikleri için düzen algoritması özelliklerini belirler. Tabloların sütunlarda (td) bir düzen aldoritması vardır ve eğer içerik boyuta sığmıyorsa otomatik olarak büyür.

değerler ;

  • auto ; otamatik değer alır.(varsayılan değerdir. Hücre içerik kadar değer alır.)
  • fixed; sabit.
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

table-layout: auto; (kelimeler arasında boşluk yok)

Örnek-2 ;

table-layout:fixed; (kelimeler arasında boşluk yok)

Örnek-3 ;

table-layout:fixed; (kelimeler arasında boşluk var)
