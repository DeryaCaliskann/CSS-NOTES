border-collapse

Html table (tablo) elementinin kenarlık / çerçeve birleştirme durumunu belirler. th ve td için farklı stil tanımlayıp collapse dersek hücrenin stili geçerli olur. Tablonun stili yok sayılır.

değerler ;

  • seperate; ayrı kenarlı /çerçeve (varsayılan)
  • collapse; bitişik kenarlık / çerçeve
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

border-collapse: separate;
