CSS kutu modeli

Html'de her element dikdörtgen şeklinde bir alan işgal eder. Html elementlerini işgal ettiği alan Css'de kutu modeli olarak isimlendirilir. Çünkü Css her Html elementini kutu olarak kabul eder.

Bir Html elementinin blok veya satır düzeyinde olması, pozisyonları, varsayılan özellikleri, gösterim türleri, renklendirilmesi vs. tüm işlemler Css sayesinde belirlenebilir.

Bir Html elementinin kutu modelinde;

  • margin; kenar boşluğu/dış boşluk
  • border; kenarlık,çerçeve
  • padding; dolgu / iç boşluk
  • content; içerik alanları bulunur.

Bir Html elementinin kutu modeline göre konum değeride bulunmaktadır;

  • top (üst)
  • right (sağ)
  • bottom (bottom)
  • left (sol)

Kutu modelini boyutları sayesinde bir Html elementinin tüm tarayıcılarda doğru şekilde görünmesi sağlanır. Hesaplama işlemi iki yapı ile sağlanır;

1-genişlik hesaplama;

Total width= width + left padding + right padding + left border + right border + left margin + right margin

2-yükseklik hesapalama;

Total height= height+ top padding + bottom padding + top border + bottom border + top margin + bottom margin
