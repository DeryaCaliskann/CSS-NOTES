html'in meta tag'ı içerisinde viewport (görünür alan)kullanimi ;

Html'in meta tag'ı içerisinde viewpot değeri kullanarak web sayfasının görünür alanını, kullanıcını kullandığı browser'ın görünür alanı olarak tanımlamak mümkündür. Görünür alan kullanıcının cihazına göre değişiklik gösterebilir. Viewport'un amacı, mobil browser'lardaki tasarım düzenini sağlamaktır. Görüntüyü zoom yapmak için shift tuşuyla beraber maouse'un sol tuşuna basarak yukarı aşağı hareket ettiriyoruz. viewport değeri head tagları arasındaki meta tagıyla belirlenir.

html'in meta tag'ı içerisinde viewport değerlerinin özellikleri için kullanılan değerler.
attribute (özellik) value (değer) description (tanım) standart value (standart değer)
width device-width pixel değer genişlik değeridir device width
height device-height pixel değer yükseklik değeridir device-height
initial-scale 0.1 - 10.0 zoom seviyesi değeridir. 1.0
minimum-scale 0.1 - 10.0 minimum zoom seviyesi değeridir. 0.1
maximum-scale 0.1 - 10.0 maksimum zoom seviyesi değeridir. 10.0
user-scalable yes-no kullanıcını zoom yapabilme yetkisi değeridir. yes
target-densitydpi device-dpi/ low-dpi/ medium-dpi-/high-dpi dpi(inç başına nokta sayısı) değeridir device-dpi
