background-repeat

html elementinin atanmış olan arkapalan resminin tekrarlanma özelliğini belirler.

değerler ;

  • repeat ; tekrar et (varsayılan)
  • space; boşluk (x ekseni boyunca)
  • round; yuvarla
  • no-repeat ; tekrar yok
  • repeat-x ; x ekseni boyunca tekrar et.
  • repeat-y; y ekseni boyunca tekrar et.
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

background-repeat: space;

Örnek-2 ;

background-repeat: round;

Örnek-3 ;

background-repeat: no-repeat;

Örnek-4 ;

background-repeat: repeat-x;

Örnek-5 ;

background-repeat: repeat-y;

Örnek-6 ;

background-image: initial;(varsayılan)

Örnek-7;

background-image: url(../img/desen-3.jpeg);
