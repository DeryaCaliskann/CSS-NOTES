background-position

html elementine atanmış olan arka plan resminin konum özellklerini belirler. Konumlandırmalar body yani gövde içerisinde ve element içerisinde farklılık gösterebilir.

değerler ;

  • length ;uzunluk değeri
  • % ;yüzde değeri
  • position ;top,right,bottom,left,center değerlerini alır.
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

background-position: 100px; (Tek parametrede soldan 100px ileri iter, yükseklik değerini ortadan ikiye kesip y ekseninde %50 ye resmi yerleştiriyor.)

Örnek-2 ;

background-position: 100px 25px;(ilk parametre x eseninde ikinci parametre y ekseninde çalışır) konumlandırır.

Örnek-3;

background-position: 25%; (element genişliğini baz alarak resmi ortalar ve %25 sola iter.)

Örnek-4 ;

background-position: 50% 80%; (ilk parametre x ekseninde ikinci parametre y ekseninde çalışır)

Örnek-5 ;

background-position: 50% 0%;

Örnek-6 ;

background-position: 50% -20%;(eksi değer elementte dışa iter. Gövdede içe iter)

Örnek-7 ;

background-position: 500px -30px;

Örnek-8 ;

background-position:top left;

Örnek-9 ;

background-position:bottom right;

Örnek-10 ;

background-position:center;

Örnek-11 ;

background-position:center left;

Örnek-12 ;

background-position:right;
