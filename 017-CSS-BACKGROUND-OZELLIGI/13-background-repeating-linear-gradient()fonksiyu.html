background :repeating-linear-gradient () fonksiyonu

Html elementinin doğrusal arkaplan renk geçişi tekrarlanma özelliklerini belirler. Her tarayıcıda doğru çalışması için prefix'ler eklenmelidir.

parametreler ;

  • Direction; yön
    • Başlangıç noktası (açı - derece)
  • Color; renk
    • başlangıç rengin - bitiş rengi
  • Dilimleme

Örnek-1 ;

background: repeating-linear-gradient(blue, yellow);

Örnek-2 ;

background: repeating-linear-gradient(blue 0%, yellow 25%);

Örnek-3 ;

background: repeating-linear-gradient(to right, blue 0%, yellow 25%);

Örnek-4 ;

background: repeating-linear-gradient( to left, red 0%, yellow 15%,blue 25%);

Örnek-5 ;

background: repeating-linear-gradient(to left top, red 0%, yellow 15%, blue 25%);

Örnek-6 ;

background: repeating-linear-gradient(45deg, red 0%, yellow 15%, blue 25%);
