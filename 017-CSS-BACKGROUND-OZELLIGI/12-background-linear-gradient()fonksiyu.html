background: linear-gradient () fonksiyonu

html elementinin doğrusal arkaplan renk geçişi özelliklerini belirler. Her tarayıcıda doğru çalışması için prefix'ler eklenmelidir.

parametreler ;

  • Direction; yön
    • Başlangıç noktası (açı - derece)
  • Color; renk
    • başlangıç rengin - bitiş rengi

Örnek-1 ;

background: linear-gradient(blue, yellow, green);

Örnek-2 ;

background: linear-gradient(to right, blue, yellow, green);

Örnek-3 ;

background: linear-gradient(to bottom right, blue, yellow, green);

Örnek-4 ;

background: linear-gradient(90deg, blue, yellow, green);

Örnek-5 ;

background: linear-gradient(28deg, blue, yellow, green);

Örnek-6 ;

background: linear-gradient(-45deg, blue, yellow, green);

Örnek-7 ;

background: linear-gradient(to right, blue 0%, yellow 40%);

Örnek-8 ;

background: linear-gradient(to top, blue 0%, yellow 20%,red 30%);

Örnek-9 ;

background: linear-gradient(to right, yellow 20%,red 30%);
