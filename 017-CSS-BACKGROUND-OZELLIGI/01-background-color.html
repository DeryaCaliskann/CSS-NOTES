background-color

html elementinin arka plan rengini belirler. Bu sayfanın gövdesi yada herhangi bir elementde olabilir.

değerler ;

  • transparent ; transparan (varsayılan değer)
  • color ; renk
  • initial ; html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; html elementi üst elementinin biçimlendirme değerini alır.

Örnek-1 ;

background-color: blueviolet;

Örnek-2 ;

background-color:rgba(0, 0, 255, 1.0) ;

Örnek-3 ;

background-color: transparent;

Örnek-4 ;

background-color: initial;

Örnek-5 ;

background-color:green;

background-color:inherit;
