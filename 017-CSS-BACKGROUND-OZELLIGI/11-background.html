background özelliği

Html elementinin arkaplan özellğini veya özelliklerini belirler.

değerler ;

  • color ; renk
  • image ; resim
  • repeat ; tekrar et
  • position ; pozisyon
  • size ; boyut
  • attachment ; sayfa hereketine göre davranış
  • origin ; resmin başlangıç konumu
  • clip ; rengin başlangıç konumu
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
  • inherit ; Html elementi üst elementinin biçimlendirme değerini alır.
