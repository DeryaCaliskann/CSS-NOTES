animation-fiil-mode özelliği

Html elementine tanımlanmış olan animasyon oynatma işlemi tamamlandığında, html elementinin konum özelliklerini belirler. Döngü olan animasyonlarda pek kullanılmaz. Prefix tanımlanır.

değerler;

  • none ; değer yok
  • forwards ; ileride
  • backwards ; geride
  • both ; her ikiside
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

animation-fill-mode: normal;

örnek-2 ;

animation-fill-mode: forwards;

örnek-3 ;

animation-fill-mode: backwards;

örnek-4 ;

animation-fill-mode:both;
