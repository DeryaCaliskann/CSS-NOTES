animation özelliği

Html elementinin tüm animasyon özelliklerini belirler. Animasyonu transition dan (geçiş) farkı, animasyon bir döngü oluşturur ve bu animasyonu biz sonalandırana kadar bitmez. Parametrelerin sırası önemlidir. Prefix tanımlanır.

parametreler ;

  • animation-name ; animasyonun adı.(@keyframes tanımlamasıyla yapılır) Varsayılan değer none
  • animation-duration ; animasyonun ne kadar sürede tamamlanacağı. Varsayılan değer 0
  • animation-timing-function; animasyonun hız eğrisi. Varsayılan değer ease
  • animation-delay ; animasyonu bekleme süresi. Varsayılan değer 0
  • animation-iteration-count; animasyonun tekrarlanma işlemi. Varsayılan değer 1
  • animation-direction ; animasyonun oynatma yönü. Varsayılan değer normal
  • animation-fill-mode ; animasyonun oynatma işlemi tamamlandığındaki konumu. Varsayılan değer none.
  • animation-play-state ; animasyonu çalışıp çalışmadığını bildirir. Varsayılan değer running

değerler ;

  • animation value ; animasyon özelliği değerleri
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

animation: ornek1 4s linear 0s infinite normal none running;

Örnek-2 ;

animation: ornek2 4s linear 0s infinite alternate none running;
