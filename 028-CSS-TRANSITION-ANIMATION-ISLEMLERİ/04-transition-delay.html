transition-delay özelliği

Html elementinine tanımlanmış olan geçiş özelliğinin kaç saniye yada kaç milisaniye sonra başlayacağını belirler. Prefix tanımlanır.

değerler ;

  • time ; zaman (saniye: s, milisamiye: ms).
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

transition-delay: 2s;
Merhaba Dünya

Örnek-2 ;

transition-delay: 2s,4s;
Merhaba Dünya

Örnek-3 ;

transition-delay: 1s, 3s, 5s, 7s;
Merhaba Dünya
