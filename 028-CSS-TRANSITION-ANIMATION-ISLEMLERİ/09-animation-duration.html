animation-duration özelliği

Html elementine tanımlanmış olan animasyon döngüsünün kaç saniye ve kaç milisaniye sonunda tamamlanacağını belirler. Prefix tanımlanır.

değerler;

  • time ; zaman (saniye:s milisaniye :ms)
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

animation-duration: 5s;

Örnek-2 ;

animation-duration: 3000ms;

Örnek-3 ;

animation-duration: 5s;
