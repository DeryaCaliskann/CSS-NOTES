animation-name özelliği

Html elementine atanmış olan animasyonun, @keyframes özelliği için tanımlanmasını belirler. Prefix tanımlanır.

değerler;

  • none ; değer yok (varsayılan)
  • @keyframes name ; animasyon aşamaları adı.
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

animation-name: none;

Örnek-2 ;

animation-name: ornek2;

Örnek-3 ;

animation-name: ornek3;
