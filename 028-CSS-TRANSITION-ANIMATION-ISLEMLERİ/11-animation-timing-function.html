animation-timing-function özelliği

Html elementine tanımlanmış olan animasyon döngüsünün hız eğrisi özelliklerini belirler . Prefix tanımlanır.

değerler;

  • ease ; yavaş olarak başlayarak hızlanır. Sona doğru yavaşlar. (Varsayılan değer)
  • linear ; başlangıçta sona kadar aynı hız.
  • ease-in ; yavaş olarak başlayarak sona doğru hızlanır.
  • ease-out ;hızlı olarak başlayarak sona doğru yavaşlar.
  • ease-in-out ; yavaş olarak başlayarak ortalarda hızlanır sona doğru tekrar yavaşlar.
  • step-start ; animasyon süresine bakılmaksızın doğrudan sona ulaşır
  • step-end ; animasyon süresine bakılarak, geçiş süresi kadar bekleyerek doğrudan sona ulaşur.
  • steps() ; başlangıç ve bitiş arasında adımlamalı animasyon işlemi yapar.(1.parametreadım sayısı 2. parametre end/start)
  • cubic-bezier; bezier eğrisi (hhtp://cubic-bezier.com)
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
