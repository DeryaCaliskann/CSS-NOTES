transition-property özelliği

Html elementinine tanımlanmış olan geçiş özelliği için kullanılması gereken css'de tanımlı olan özellik adını belirler. Prefix tanımlanır.

değerler ;

  • all ; hepsi.
  • property ; özellik.
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

transition: 3s;
transition-property: all;

Örnek-2 ;

transition: 3s;
transition-property: width;

Örnek-3 ;

transition: 3s;
transition-property: width,height;

Örnek-4 ;

transition: 1s, 3s;
transition-property: width,height;

Örnek-5 ;

transition: 5s;
transition-property: transform;
transform:rotate(1800deg);
