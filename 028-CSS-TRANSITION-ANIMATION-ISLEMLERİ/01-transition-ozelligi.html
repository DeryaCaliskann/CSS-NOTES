transition özelliği

Html elementinin tüm geçiş özelliklerini belirler. Prefix tanımlamak gerekir.

parametreler ;

  • transition-property ; özellik adı
  • transition-duration ; geçiş süresi
  • transition-timing-function ; hız eğrisi
  • transition-delay ; duraksama süresi

değerler ;

  • property value; özellik değeri
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.
