transition-duration özelliği

Html elementinine tanımlanmış olan geçiş özelliğinin kaç saniye veya kaç milisaniye sonunda tamamlanacağını belirler. Prefix tanımlanır.

değerler ;

  • time ; zaman (saniye: s, milisamiye: ms).
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

Örnek-1 ;

transition-property:all;
transition-duration: 3s;
Merhaba Dünya

Örnek-2 ;

transition-property: width,color;
transition-duration: 3s,1ms;
Merhaba Dünya

Örnek-3 ;

transition-property: transform;
transition-duration: 2000ms;
transform: skew(-20deg);
Merhaba Dünya

Örnek-4 ;

transition-property: width, transform;
transition-duration: 100ms, 3000ms;
Merhaba Dünya
