animation-direction özelliği

Html elementine tanımlanmış olan animasyonun oynatma yönü özelliklerini belirler. Prefix tanımlanır.

değerler;

  • normal ; normal
  • reverse ; ters
  • altarnate ; alternatif (oynatma bittiğinde kesiklik olmaz)
  • alternate reverse ; ters alternatif
  • initial ; Html elementi varsayılan biçimlendirme değerine döner.

örnek-1 ;

animation-direction: normal;
Merhaba Dünya

örnek-2 ;

animation-direction: alternate;
Merhaba Dünya

örnek-3 ;

animation-direction: reverse;
Merhaba Dünya

örnek-4 ;

animation-direction: alternate-reverse;
Merhaba Dünya
