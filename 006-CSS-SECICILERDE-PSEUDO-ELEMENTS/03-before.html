3-before pseudo element

Html elementinin öncesine içerik atayarak stil tanımlar.

Not:Html Unicode sembollerinide before sahte elementiyle ekleyebiliriz.

tag'larınada before sahte elementini kullanarak içerik atayabiliriz.

Örnek ;

::before{
font-size: 25px;
}

p.deneme::before {
content: "Belirtilen ";
color: green;
}

p#ornek::before{
content: "\2600 \2601 \2602 \2603 ";
}

button::before{
content: "\2705";
}
