Tanım : CSS seçicilerinde kullanılan beş farklı pseudo element türü bulunmaktadır.Bu sayede istenilen bir veya daha fazla elementin üzerinde biçimlendirme işlemleri yapılabilir.

1-first-letter pseudo element

Belirtilen Html elementininin ilk karakterine stil tanımlar.

Örnek ;

h1::first-letter {
color: red;
}
