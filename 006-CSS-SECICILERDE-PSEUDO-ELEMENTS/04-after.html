4-after pseudo element

Html elementinin sonrasına içerik atayarak stil

Not:Html Unicode sembollerinide after sahte elementiyle ekleyebiliriz.

tag'larınada after sahte elementini kullanarak içerik atayabiliriz.

Örnek ;

::after{
font-size: 25px;
}

p.deneme::after {
content: " tanımlar.";
color: green;
}

p#ornek::after{
content: "\2600 \2601 \2602 \2603 ";
}

button::after{
content: "\2705";
}
