Css'de Browserler İçin PREFİX (ÖNEK) Kullanımı :

Hemen hemen her browser'in geliştirici firması ve browser visualization (tarayıcı görüntüleme motorları) farklı olduğu için, bazı durumlarda Css özelliklerine prefix tanımlanmalıdır. Css özelliklerinin tüm browser'lar tarafından algılanarak yorumlanabilmesi ve kusursuz şekilde çalıştırılabilmesini prefix tanımları sağlar.

  • Internet Explorer / -ms-
  • Microsoft Edge / -ms-
  • Microsoft Office / -mso- (posta içermiyorsa kullanılması gerekmez)
  • Mozilla Firefox / -moz-
  • Google Chorome / -webkit-
  • Yandex Browser / -webkit-
  • Safari / -webkit-
  • Opera / -o-

Css özellikleri için prefix tanımlarının ,W3C tarafından belirtilmiş en doğru sıralaması aşağıdaki şekildedir.

  • -webkit-
  • -moz-
  • -o-
  • -ms-
  • -mso-(gerekli ise)
  • standart kodlama

NOT: Border-radius, animasyon, opacity (şeffaflık) kullanılacaksa prefix kullanmalıyız.
