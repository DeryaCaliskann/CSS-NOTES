list-style özelliğinin kullanımı;

Listedeki madde işaretçisinin tüm özelliklerini belirler.

parametreler;

  • list-style-type ;listedeki madde işaretçisinin türü.
  • list-style-position ;listedeki madde işaretçisinin içerik akışındaki konumu
  • list-style-image ;listedeki madde işaretçisinin resmi
  • initial; html elementi varsayılan biçimlendirme değerine döner
  • inherit; html elementi üst elementini biçimlendirme değerini alır

Örnek-1 ;

list-style:square;
